GIP — Sportclub Ledenportaal

Bouw een volledige full-stack webapplicatie: leden kunnen inloggen en inschrijven voor activiteiten, de admin beheert het programma. Je combineert Node.js, MySQL, bcrypt en front-end JavaScript in één project.

🚀 Wat bouw je?

Het Sportclub Ledenportaal is een webapplicatie met vier soorten pagina’s:

  • Startpagina — alle activiteiten zichtbaar voor iedereen
  • Register / Login — account aanmaken en inloggen
  • Dashboard — inschrijven voor activiteiten (beschermd: leden only)
  • Adminpanel — activiteiten beheren (beschermd: admin only)
