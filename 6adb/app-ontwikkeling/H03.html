3.1 Template literals

Tot nu toe combineerde je strings met de +-operator (concatenatie):

JavaScript
let naam = "Lena";
let leeftijd = 17;
let zin = "Hallo, mijn naam is " + naam + " en ik ben " + leeftijd + " jaar oud.";
console.log(zin);

Dit werkt, maar wordt al snel onleesbaar bij langere zinnen. Template literals zijn de moderne oplossing.

Syntax

Gebruik backticks (`) in plaats van aanhalingstekens. Variabelen en uitdrukkingen zet je tussen ${ }:

JavaScript
let naam = "Lena";
let leeftijd = 17;

let zin = `Hallo, mijn naam is ${naam} en ik ben ${leeftijd} jaar oud.`;
console.log(zin);
// "Hallo, mijn naam is Lena en ik ben 17 jaar oud."

Tussen ${ } kan je elke JavaScript-expressie plaatsen:

JavaScript
let a = 10;
let b = 3;

console.log(`${a} + ${b} = ${a + b}`);         // "10 + 3 = 13"
console.log(`${a} * ${b} = ${a * b}`);         // "10 * 3 = 30"
console.log(`Is a groter? ${a > b}`);           // "Is a groter? true"

Multiline strings

Met backticks kan je strings over meerdere regels schrijven — zonder \n:

JavaScript
let berichtNieuw = `Beste leerling,
Welkom in de les.
Veel succes!`;

console.log(berichtNieuw);

3.2 Waarom template literals?

String concatenatieTemplate literal
LeesbaarheidMoeilijk bij veel variabelenVeel beter — variabelen staan in de tekst
BerekeningenMoet buiten de stringKan rechtstreeks in ${ }
Meerdere regelsVereist \nGewoon enter drukken
FoutenMakkelijk een + vergetenMinder typfouten
💡 Vuistregel

Gebruik template literals wanneer je variabelen in een string wil verwerken. Gebruik gewone aanhalingstekens voor eenvoudige strings zonder variabelen.

3.3 Lengte, hoofdletters, kleine letters

.length — het aantal tekens

JavaScript
let woord = "Antwerpen";
console.log(woord.length); // 9

// Handig: controleer of een invoer lang genoeg is
let wachtwoord = "abc";
if (wachtwoord.length < 8) {
  console.log("Wachtwoord is te kort!");
}

.toUpperCase() en .toLowerCase()

JavaScript
let tekst = "Hello World";

console.log(tekst.toUpperCase()); // "HELLO WORLD"
console.log(tekst.toLowerCase()); // "hello world"

// Praktisch: vergelijken zonder rekening met hoofdletters
let invoer = "BRUSSEL";
if (invoer.toLowerCase() === "brussel") {
  console.log("Correcte stad!");
}

3.4 includes, startsWith, endsWith

.includes(zoekterm) — zit dit er in?

JavaScript
let zin = "JavaScript is een programmeertaal.";

console.log(zin.includes("JavaScript")); // true
console.log(zin.includes("Python"));     // false

let email = "lena.pieters@student.be";
if (email.includes("@")) {
  console.log("Geldig e-mailadres (bevat @)");
}

.startsWith(tekst) — begint ermee?

JavaScript
let url = "https://www.voorbeeld.be";
if (url.startsWith("https")) {
  console.log("Beveiligde verbinding");
}

.endsWith(tekst) — eindigt ermee?

JavaScript
let bestand = "cursus.pdf";

console.log(bestand.endsWith(".pdf"));  // true
console.log(bestand.endsWith(".docx")); // false

if (bestand.endsWith(".pdf")) {
  console.log("Dit is een PDF-bestand");
}

3.5 slice en trim

.slice(start, einde) — een deel ophalen

JavaScript
let woord = "programmeren";

console.log(woord.slice(0, 7));  // "program"
console.log(woord.slice(7));     // "meren"
console.log(woord.slice(-5));    // "meren"

// Gebruikersnaam genereren uit e-mailadres
let email = "lena.pieters@school.be";
let gebruikersnaam = email.slice(0, email.indexOf("@"));
console.log(gebruikersnaam); // "lena.pieters"

.trim() — spaties verwijderen

JavaScript
let invoer = "   Hallo wereld   ";

console.log(invoer.trim());      // "Hallo wereld"
console.log(invoer.trimStart()); // "Hallo wereld   "
console.log(invoer.trimEnd());   // "   Hallo wereld"

// Formulierinvoer opschonen
let naam = "  Bo  ";
let opgeschoond = naam.trim().toLowerCase();
console.log(opgeschoond); // "bo"

3.6 padStart, padEnd, repeat

.padStart(lengte, karakter) — links opvullen

JavaScript
let getal = "42";
console.log(getal.padStart(5, "0")); // "00042"

// Praktisch: getallen uitlijnen
for (let i = 1; i <= 5; i++) {
  console.log(String(i).padStart(3, "0") + ": item");
}
// 001: item, 002: item, ...

.padEnd(lengte, karakter) — rechts opvullen

JavaScript
let naam = "Ali";
console.log(naam.padEnd(10, ".")); // "Ali......."

.repeat(n) — herhalen

JavaScript
console.log("ha".repeat(3));    // "hahaha"
console.log("=-".repeat(20));   // scheidingslijn
console.log("*".repeat(5));     // "*****"

3.7 split en join

.split(scheidingsteken) — string opdelen in array

JavaScript
let zin = "appels,brood,melk,kaas";
let producten = zin.split(",");
console.log(producten); // ["appels", "brood", "melk", "kaas"]

let tekst = "Hallo wereld hoe gaat het";
let woorden = tekst.split(" ");
console.log(woorden); // ["Hallo", "wereld", "hoe", "gaat", "het"]

.join(scheidingsteken) — array samenvoegen tot string

JavaScript
let producten = ["appels", "brood", "melk"];

console.log(producten.join(", ")); // "appels, brood, melk"
console.log(producten.join(" en ")); // "appels en brood en melk"

// Woord omdraaien
let woord = "programmeer";
let omgekeerd = woord.split("").reverse().join("");
console.log(omgekeerd); // "reermargorp"

ADB09 Oefening 1 — Naamkaartje-generator

Bouw een naamkaartje-generator die automatisch een naamkaartje opmaakt. Gebruik template literals en minimaal 5 verschillende string methodes.

JavaScript
// Invoergegevens (gesimuleerde formulierinvoer)
let voornaam = "  lena  ";
let familienaam = "  PIETERS  ";
let school = "Go! Atheneum Munsterbilzen";
let richting = "data- en applicatiebeheer";
let leerjaar = 6;
  1. Schoon de voornaam en familienaam op met trim()
  2. Formatteer de namen zodat de eerste letter groot is en de rest klein
  3. Genereer een gebruikersnaam: voornaam.familienaam (alles lowercase, geen spaties)
  4. Opmaken van een kaartje met padEnd() voor uitlijning en repeat() voor scheidingslijnen
  5. Controleer of de school begint met "Go!" en of de richting "data" bevat
ADB09 / ADB10 Oefening 2 — E-mailvalidator

Schrijf een functie valideerEmail(email) die controleert of een e-mailadres geldig is. Gebruik string methodes (geen reguliere expressies).

  1. Controleer of het adres een @ bevat met includes()
  2. Controleer of het adres eindigt op .be, .com of .org met endsWith()
  3. Genereer een gebruikersnaam door alles voor de @ te pakken met slice() en indexOf()
  4. Schoon het adres op met trim() en zet het om naar kleine letters met toLowerCase()
  5. Toon het resultaat als template literal: Email "..." is geldig/ongeldig. Gebruikersnaam: "..."
ADB09 Oefening 3 — CSV-verwerker

Je ontvangt data als een CSV-string (comma separated values). Verwerk deze naar bruikbare JavaScript-data.

JavaScript
let csvData = "Ali,72,6A;Bo,45,6B;Caro,88,6A;Daan,38,6B";
  1. Splits de string op ; om een array van rijen te krijgen
  2. Splits elke rij op , om naam, punten en klas te scheiden
  3. Maak van elk rij-array een object {naam, punten, klas}
  4. Log de naam en punten van elke student via een template literal
  5. Sla de namen van alle studenten op in een array en gebruik join(" | ") om ze als één string te loggen
