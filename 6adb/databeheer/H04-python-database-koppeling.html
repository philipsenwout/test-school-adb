1. mysql-connector-python

De officiële MySQL-bibliotheek voor Python heet mysql-connector-python. We installeren ze eenmalig:

# Installeren via pip
pip install mysql-connector-python

2. Verbinding en cursor

import mysql.connector

# Verbinding maken
verbinding = mysql.connector.connect(
    host     = "localhost",
    user     = "root",
    password = "jouw_wachtwoord",
    database = "winkel_db"
)

# Cursor aanmaken (voert queries uit)
cursor = verbinding.cursor()

# ... queries uitvoeren ...

# Altijd afsluiten!
cursor.close()
verbinding.close()
Tip — with-statement voor de verbinding

Je kan ook een context manager gebruiken, maar mysql-connector ondersteunt dit niet standaard. Gebruik liever een try/finally-blok of maak een aparte functie die de verbinding beheert. Sluit altijd de cursor en verbinding af in het finally-blok.

# Betere aanpak: try/finally
verbinding = None
cursor = None
try:
    verbinding = mysql.connector.connect(
        host="localhost", user="root",
        password="geheim", database="winkel_db"
    )
    cursor = verbinding.cursor(dictionary=True)  # geeft dicts terug
    # ... code hier ...
except mysql.connector.Error as fout:
    print(f"Databasefout: {fout}")
finally:
    if cursor:    cursor.close()
    if verbinding: verbinding.close()

3. SELECT-queries uitvoeren

cursor = verbinding.cursor(dictionary=True)

# Alle producten ophalen
cursor.execute("SELECT * FROM producten")
producten = cursor.fetchall()  # lijst van dicts

for p in producten:
    print(f"{p['naam']}: €{p['prijs']}")

# Één rij ophalen
cursor.execute("SELECT * FROM producten WHERE product_id = 1")
product = cursor.fetchone()   # dict of None
if product:
    print(product['naam'])

# Aantal rijen ophalen
cursor.execute("SELECT COUNT(*) AS totaal FROM producten")
resultaat = cursor.fetchone()
print(f"Aantal producten: {resultaat['totaal']}")
MethodeTeruggeeftGebruik
fetchone()Één rij (of None)Als je max. 1 resultaat verwacht
fetchall()Lijst van alle rijenVoor meerdere resultaten
fetchmany(n)Lijst van n rijenVoor paginering

4. INSERT, UPDATE en DELETE

# INSERT
cursor.execute(
    "INSERT INTO producten (naam, prijs, categorie) VALUES (%s, %s, %s)",
    ("Toetsenbord", 89.99, "Accessoires")
)
verbinding.commit()  # verplicht voor wijzigingen!
print(f"Ingevoegd met ID: {cursor.lastrowid}")

# UPDATE
cursor.execute(
    "UPDATE producten SET prijs = %s WHERE product_id = %s",
    (79.99, cursor.lastrowid)
)
verbinding.commit()
print(f"Gewijzigde rijen: {cursor.rowcount}")

# DELETE
cursor.execute(
    "DELETE FROM producten WHERE product_id = %s",
    (cursor.lastrowid,)
)
verbinding.commit()

5. Geparametriseerde queries en SQL-injectie

Wat is SQL-injectie?

SQL-injectie is een aanval waarbij kwaadaardige SQL-code in een invoerveld wordt geplaatst om ongeautoriseerde toegang te krijgen tot de database. Voorbeeld: als je WHERE naam = '" + invoer + "'" schrijft en de gebruiker geeft ' OR '1'='1 in, geeft de query alle records terug!

# FOUT: string-concatenatie (kwetsbaar voor SQL-injectie!)
naam = input("Productnaam: ")
cursor.execute("SELECT * FROM producten WHERE naam = '" + naam + "'")

# GOED: geparametriseerde query (veilig)
naam = input("Productnaam: ")
cursor.execute("SELECT * FROM producten WHERE naam = %s", (naam,))
# MySQL vervangt %s door de veilig-gesanitiseerde waarde

# Meerdere parameters
cursor.execute(
    "SELECT * FROM producten WHERE categorie = %s AND prijs < %s",
    ("Elektronica", 500)
)

Let op: gebruik altijd %s als placeholder, ook voor getallen. De connector zorgt voor de juiste escaping.

6. Transactions

Een transactie groepeert meerdere queries zodat ze samen slagen of samen mislukken (alles of niets). Dit is cruciaal voor data-integriteit.

try:
    # Zet auto-commit uit (standaard al uit in mysql-connector)
    verbinding.start_transaction()

    # Stap 1: bestelling aanmaken
    cursor.execute(
        "INSERT INTO bestellingen (klant_id, besteldatum) VALUES (%s, CURDATE())",
        (klant_id,)
    )
    nieuwe_bestelling_id = cursor.lastrowid

    # Stap 2: bestellingsregels toevoegen
    for product_id, aantal in winkelwagen:
        cursor.execute(
            "INSERT INTO bestelling_regels (bestelling_id, product_id, aantal) VALUES (%s, %s, %s)",
            (nieuwe_bestelling_id, product_id, aantal)
        )

        # Stap 3: voorraad verminderen
        cursor.execute(
            "UPDATE producten SET voorraad = voorraad - %s WHERE product_id = %s",
            (aantal, product_id)
        )

    verbinding.commit()  # alles gelukt, definitief opslaan
    print("Bestelling succesvol geplaatst!")

except mysql.connector.Error as fout:
    verbinding.rollback()  # ongedaan maken bij fout
    print(f"Fout: {fout} — bestelling ongedaan gemaakt")

Samenvatting

Samenvatting
  • mysql.connector.connect() maakt verbinding. Altijd afsluiten in finally.
  • cursor(dictionary=True) geeft resultaten als dictionaries terug.
  • fetchone() = één rij; fetchall() = alle rijen.
  • Gebruik altijd %s-placeholders (NOOIT string-concatenatie) om SQL-injectie te vermijden.
  • Na INSERT/UPDATE/DELETE: verbinding.commit() is verplicht.
  • Transactions: start_transaction() + commit() of rollback() bij fout.
