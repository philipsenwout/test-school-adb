Raspberry Pi hardware (verdieping)

LD-ES-05Raspberry Pi hardware verdiept gebruiken — verdieping
🎯 Leerdoelen
  • De leerling kan de verschillen tussen Raspberry Pi 4 en Pi 5 benoemen en beargumenteren.
  • De leerling kan GPIO-pinnen configureren voor PWM, I2C en SPI communicatie.
  • De leerling kan I2C-apparaten detecteren en aansturen via Python.
  • De leerling kan SPI-communicatie opzetten met spidev.
  • De leerling kan een Raspberry Pi headless instellen en verbinden via SSH.

2. GPIO verdieping

In 5ADB leerden we de basispin-functies. Nu gaan we dieper:

Pin-typeFunctiePinnen (BCM)
GPIODigitale input/output2–27 (niet allemaal)
PWMPulsbreedte modulatie (analoge output simuleren)18, 19 (hardware PWM)
I2CInter-Integrated Circuit protocolSDA=2, SCL=3
SPISerial Peripheral InterfaceMOSI=10, MISO=9, SCLK=11, CE0=8
UARTSeriële communicatieTX=14, RX=15
 
import RPi.GPIO as GPIO
import time

GPIO.setmode(GPIO.BCM)

# PWM op pin 18 (hardware PWM)
GPIO.setup(18, GPIO.OUT)
pwm = GPIO.PWM(18, 1000)   # 1000 Hz frequentie
pwm.start(0)               # 0% duty cycle (uit)

# LED dimmen van 0% naar 100%
for duty in range(0, 101, 5):
    pwm.ChangeDutyCycle(duty)
    time.sleep(0.05)

pwm.stop()
GPIO.cleanup()

3. I2C protocol

I2C (Inter-Integrated Circuit) is een seriëel communicatieprotocol waarmee meerdere apparaten op twee draden kunnen communiceren: SDA (data) en SCL (klok). Elk apparaat heeft een uniek adres (bv. 0x48).

# I2C inschakelen: sudo raspi-config → Interface Options → I2C → Enable

# Aangesloten I2C-apparaten detecteren
sudo i2cdetect -y 1

# Python met smbus2 library
# pip install smbus2
import smbus2

bus = smbus2.SMBus(1)   # I2C bus 1 (SDA=GPIO2, SCL=GPIO3)

ADRES = 0x48  # adres van jouw sensor (zie i2cdetect)

# Byte lezen van een register
waarde = bus.read_byte_data(ADRES, 0x00)
print(f"Register 0x00: {waarde}")

# Meerdere bytes lezen (bv. 2 bytes voor een 16-bit waarde)
bytes_data = bus.read_i2c_block_data(ADRES, 0x00, 2)
waarde_16bit = (bytes_data[0] << 8) | bytes_data[1]
print(f"16-bit waarde: {waarde_16bit}")

4. SPI protocol

SPI (Serial Peripheral Interface) is sneller dan I2C en gebruikt 4 draden: MOSI (Master Out Slave In), MISO (Master In Slave Out), SCLK (klok) en CS/CE (Chip Select).

# SPI inschakelen: sudo raspi-config → Interface Options → SPI → Enable

# pip install spidev
import spidev

spi = spidev.SpiDev()
spi.open(0, 0)           # bus 0, device 0 (CE0)
spi.max_speed_hz = 1000000  # 1 MHz

# Byte sturen en ontvangen (full duplex)
antwoord = spi.xfer2([0x01, 0x02, 0x03])
print(f"Ontvangen: {antwoord}")

spi.close()

5. Camera module

De Raspberry Pi Camera Module V2 (of V3) sluit aan via de CSI-poort.

# Camera inschakelen: sudo raspi-config → Interface Options → Camera

# Foto nemen via terminal
libcamera-still -o foto.jpg

# Video opnemen (10 seconden)
libcamera-vid -t 10000 -o video.h264

# Python via picamera2 (Raspberry Pi OS Bookworm)
# pip install picamera2
from picamera2 import Picamera2
import time

camera = Picamera2()
camera.start()
time.sleep(2)  # even laten opwarmen

camera.capture_file("foto.jpg")
print("Foto opgeslagen als foto.jpg")

camera.stop()

6. Headless setup

Een headless Raspberry Pi draait zonder monitor, muis of toetsenbord. Je beheert hem via SSH.

# 1. SSH inschakelen: maak een leeg bestand 'ssh' aan op de boot-partitie
touch /boot/ssh

# 2. WiFi configureren (wpa_supplicant.conf op boot-partitie)
# Bestand: /boot/wpa_supplicant.conf
country=BE
ctrl_interface=DIR=/var/run/wpa_supplicant GROUP=netdev
update_config=1

network={
    ssid="JouwWifiNaam"
    psk="JouwWifiWachtwoord"
}

# 3. Verbinden via SSH
ssh pi@raspberrypi.local
# of via IP: ssh pi@192.168.1.x

# 4. IP-adres vinden (vanop de Pi zelf)
hostname -I

# 5. Beveiliging: wachtwoord wijzigen (ALTIJD doen!)
passwd
Tip — Raspberry Pi Imager

Gebruik de Raspberry Pi Imager om het OS te installeren. In de geavanceerde opties kan je direct SSH inschakelen, WiFi-gegevens invullen en een gebruikersnaam/wachtwoord instellen — dan is geen handmatige headless-configuratie nodig.

Samenvatting

Samenvatting
  • Raspberry Pi 5 is sneller dan de Pi 4, maar heeft dezelfde 40-pin GPIO-header.
  • PWM simuleert analoge output op digitale pinnen (duty cycle = percentage aan-tijd).
  • I2C: 2 draden (SDA, SCL), meerdere apparaten via unieke adressen. Detecteer met i2cdetect.
  • SPI: 4 draden, sneller dan I2C, punt-tot-punt via chip select.
  • Camera module: via CSI, aansturen met libcamera-still of picamera2.
  • Headless setup: SSH + WiFi-config op boot-partitie of via Raspberry Pi Imager.
