In dit laatste hoofdstuk verbinden we alles wat we geleerd hebben. De Raspberry Pi verzamelt data, slaat die op in een database, en we analyseren en visualiseren de resultaten met Python — een volledige data pipeline van sensor naar dashboard.

1. Van SQLite naar MySQL/MariaDB

SQLite is ideaal voor kleine projecten op de Pi zelf. Voor grotere projecten of als meerdere apparaten dezelfde database delen, gebruik je MySQL of MariaDB op een centrale server.

SQLite vs MySQL — wanneer welk?
EigenschapSQLiteMySQL/MariaDB
InstallatieGeen (ingebouwd)Server nodig
Meerdere gebruikersBeperktJa
Meerdere apparatenNee (lokaal bestand)Ja (netwerk)
GebruikPrototype, Pi-projectProductie, school, bedrijf

MariaDB installeren op de Pi:

# MariaDB installeren
sudo apt install mariadb-server python3-mysqldb -y

# MariaDB starten en beveiligen
sudo systemctl start mariadb
sudo mysql_secure_installation

# Aanmelden als root
sudo mysql

# Database en gebruiker aanmaken
CREATE DATABASE pidata;
CREATE USER 'piuser'@'localhost' IDENTIFIED BY 'geheim123';
GRANT ALL PRIVILEGES ON pidata.* TO 'piuser'@'localhost';
FLUSH PRIVILEGES;
EXIT;

2. Python + MySQL koppelen

We gebruiken de bibliotheek mysql-connector-python (of PyMySQL) om vanuit Python met MySQL te praten:

# Installeer mysql-connector
pip install mysql-connector-python
# database_mysql.py
import mysql.connector
import datetime

DBCONFIG = {
    "host":     "localhost",
    "user":     "piuser",
    "password": "geheim123",
    "database": "pidata"
}

def init_db():
    conn = mysql.connector.connect(**DBCONFIG)
    cursor = conn.cursor()
    cursor.execute("""
        CREATE TABLE IF NOT EXISTS metingen (
            id          INT AUTO_INCREMENT PRIMARY KEY,
            tijdstip    DATETIME     NOT NULL,
            temperatuur DECIMAL(5,2) NOT NULL,
            locatie     VARCHAR(50)  DEFAULT 'Pi'
        )
    """)
    conn.commit()
    cursor.close()
    conn.close()

def voeg_meting_toe(temperatuur, locatie="Pi"):
    conn = mysql.connector.connect(**DBCONFIG)
    cursor = conn.cursor()
    cursor.execute(
        "INSERT INTO metingen (tijdstip, temperatuur, locatie) VALUES (%s, %s, %s)",
        (datetime.datetime.now(), temperatuur, locatie)
    )
    conn.commit()
    cursor.close()
    conn.close()

def haal_metingen_op(limit=100):
    conn = mysql.connector.connect(**DBCONFIG)
    cursor = conn.cursor(dictionary=True)
    cursor.execute(
        "SELECT tijdstip, temperatuur, locatie FROM metingen ORDER BY tijdstip DESC LIMIT %s",
        (limit,)
    )
    resultaat = cursor.fetchall()
    cursor.close()
    conn.close()
    return resultaat
Parameterised queries

Gebruik altijd %s placeholders met een tuple als tweede argument — nooit strings samenplakken met +. Dit voorkomt SQL-injectie.

# GOED
cursor.execute("INSERT INTO t (naam) VALUES (%s)", (naam,))

# FOUT (SQL-injectie mogelijk!)
cursor.execute("INSERT INTO t (naam) VALUES ('" + naam + "')")

3. Data analyseren met pandas

pandas is een Python-bibliotheek voor data-analyse. We kunnen de metingen uit de database laden als een DataFrame en er direct mee rekenen:

pip install pandas
# analyse.py
import pandas as pd
import mysql.connector

DBCONFIG = {
    "host": "localhost", "user": "piuser",
    "password": "geheim123", "database": "pidata"
}

def laad_dataframe():
    conn = mysql.connector.connect(**DBCONFIG)
    df = pd.read_sql(
        "SELECT tijdstip, temperatuur FROM metingen ORDER BY tijdstip",
        conn,
        parse_dates=["tijdstip"]
    )
    conn.close()
    return df

df = laad_dataframe()

# Basisstatistieken
print(df["temperatuur"].describe())
# count, mean, std, min, 25%, 50%, 75%, max

# Gemiddelde per dag
df["dag"] = df["tijdstip"].dt.date
daggemiddelden = df.groupby("dag")["temperatuur"].mean()
print(daggemiddelden)

# Filteren: enkel metingen boven 24 °C
warme_metingen = df[df["temperatuur"] > 24.0]
print(f"Aantal warme metingen: {len(warme_metingen)}")

4. Visualisatie met matplotlib

matplotlib maakt grafieken van onze data. Samen met pandas is dit een krachtige combinatie:

pip install matplotlib
# visualisatie.py
import pandas as pd
import matplotlib.pyplot as plt
import matplotlib.dates as mdates
from analyse import laad_dataframe

df = laad_dataframe()

# --- Grafiek 1: tijdreeks ---
fig, ax = plt.subplots(figsize=(10, 4))
ax.plot(df["tijdstip"], df["temperatuur"],
        color="#6c63ff", linewidth=1.5, label="Temperatuur")
ax.set_title("Temperatuurverloop")
ax.set_xlabel("Tijdstip")
ax.set_ylabel("Temperatuur (°C)")
ax.xaxis.set_major_formatter(mdates.DateFormatter("%d/%m %H:%M"))
fig.autofmt_xdate()
ax.legend()
plt.tight_layout()
plt.savefig("static/grafiek_tijdreeks.png", dpi=100)
plt.close()

# --- Grafiek 2: histogram ---
fig, ax = plt.subplots(figsize=(6, 4))
ax.hist(df["temperatuur"], bins=15, color="#6c63ff", edgecolor="white")
ax.set_title("Verdeling temperaturen")
ax.set_xlabel("Temperatuur (°C)")
ax.set_ylabel("Aantal metingen")
plt.tight_layout()
plt.savefig("static/grafiek_histogram.png", dpi=100)
plt.close()

print("Grafieken opgeslagen in static/")

# app.py — dashboard route
import subprocess

@app.route("/dashboard")
def dashboard():
    # Grafieken opnieuw genereren
    subprocess.run(["python3", "visualisatie.py"], check=True)
    return render_template("dashboard.html")

5. Eindproject opzet

Je eindproject combineert alles uit E-STEM 6ADB in één samenhangend systeem:

Eindproject architectuur
Raspberry Pi
├── sensor_log.py      ← cron job: elke 5 min meting loggen
├── app.py             ← Flask webserver (systemd-service)
├── database_mysql.py  ← MySQL-koppeling
├── visualisatie.py    ← grafieken genereren
├── templates/
│   ├── index.html     ← overzicht + formulier
│   └── dashboard.html ← grafieken
└── static/
    ├── style.css
    └── *.png          ← gegenereerde grafieken

MySQL/MariaDB
└── pidata.metingen    ← alle sensordata

Projectstructuur aanmaken:

# Alles in één keer installeren
pip install flask mysql-connector-python pandas matplotlib

# Projectmap aanmaken
mkdir ~/eindproject
cd ~/eindproject
mkdir templates static

# Python-bestanden aanmaken
touch app.py database_mysql.py sensor_log.py visualisatie.py

Oefeningen

Oefening 4.1 — MySQL koppeling
  • a. Installeer MariaDB op je Pi en maak de database pidata aan met gebruiker piuser.
  • b. Schrijf database_mysql.py met functies om metingen toe te voegen en op te halen.
  • c. Schrijf een testscript dat 10 willekeurige temperatuurmetingen toevoegt en ze daarna ophaalt.
  • d. Verifieer de data via de MySQL-commandoregel: SELECT * FROM metingen;
Oefening 4.2 — Dashboard met grafieken
  • a. Laad de metingen uit MySQL in een pandas DataFrame.
  • b. Maak een tijdreeksgrafiek van de temperatuur over de laatste 24 uur.
  • c. Voeg de grafiek toe aan een Flask-dashboard (/dashboard).
  • d. Voeg een tabel toe met de basisstatistieken (min, max, gemiddelde) berekend via pandas.
