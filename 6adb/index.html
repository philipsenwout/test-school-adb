GIP — Geïntegreerd Informaticaproject

Het GIP is het sluitstuk van je opleiding. Je combineert alles wat je dit jaar geleerd hebt in één zelfstandig project: een volledige webapplicatie met front-end, back-end en een verzorgde oplevering via GitHub.

Netwerken
Je deployt je GIP op een lokale server of via GitHub Pages. Je configureert de hostingomgeving en zorgt dat de applicatie bereikbaar is.
App-ontwikkeling
De volledige front-end: HTML, CSS en JavaScript. Je bouwt een responsieve, interactieve gebruikersinterface die data ophaalt uit de back-end.
Databeheer
De back-end en database. Je ontwerpt een ERD, schrijft SQL-queries en koppelt je database aan de front-end via een API of Node.js.
E-STEM
Optioneel: koppel een Raspberry Pi aan je applicatie als IoT-component, of gebruik Python-scripts als gegevensverwerker in de back-end pipeline.
GitHub
Het GIP leeft volledig in Git. Je werkt met branches, dient elke mijlpaal in als Pull Request en ontvangt feedback via code review van je leerkracht.
