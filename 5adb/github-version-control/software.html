📄 GitHub Version Control 5ADB

Git & GitHub installatie

Installeer Git op je computer en maak een gratis GitHub account aan. Volg de stappen voor jouw besturingssysteem.

📌 Wat installeren we?
  • Git — de versiebeheersoftware die lokaal op je computer draait
  • GitHub account — een gratis online platform om je code op te slaan en te delen
G

Git

Gratis en open-source versiebeheersysteem — Windows, macOS en Linux

Git is de software die bijhoudt welke wijzigingen je aanbrengt in je bestanden. Het draait lokaal op je computer. GitHub is de online dienst waar je je code mee synchroniseert.

Windows

1
Download Git voor Windows

Ga naar git-scm.com en klik op de grote downloadknop. Het detecteert Windows automatisch en biedt de nieuwste versie aan.

2
Voer de installer uit

Dubbelklik op het .exe bestand. Klik bij elke stap op Next. Pas de volgende opties aan:
✓ "Git Bash Here" — handig: rechtermuisklik in een map opent de terminal
✓ "Use Git from the command line and also from 3rd-party software"
✓ "Use Visual Studio Code as Git's default editor" (als je VS Code hebt)

3
Klik op Install en daarna Finish

De installatie duurt een minuutje. Vink "Launch Git Bash" aan bij Finish om meteen te testen.

4
Controleer de installatie

Open Git Bash (via het startmenu of rechtermuisklik in een map) en typ:

Bash
git --version

Je ziet zoiets als git version 2.47.0.windows.2. Als je dit ziet, is Git correct geïnstalleerd.

macOS

1
Controleer of Git al aanwezig is

Open Terminal (via Spotlight: Cmd+Spatie → type "Terminal") en typ:

Bash
git --version

macOS vraagt soms automatisch om de Xcode Command Line Tools te installeren. Klik op Install als dit verschijnt — dit installeert Git automatisch.

2
Git via Homebrew (alternatief)

Als je Homebrew hebt, kun je de meest recente versie van Git installeren via:

Bash
brew install git

Linux (Ubuntu/Debian)

1
Installeer via de package manager
Bash
sudo apt update
sudo apt install git
⚙

Git configureren

Stel je naam en e-mailadres in — dit staat bij elke commit

Eén keer uitvoeren op je computer — Git onthoudt deze instellingen voor alle toekomstige commits.

1
Stel je naam in
Bash
git config --global user.name "Jouw Naam"

Vervang Jouw Naam door je echte naam (bv. "Jan Peeters"). Deze naam verschijnt bij elke commit die je maakt.

2
Stel je e-mailadres in
Bash
git config --global user.email "jouw@email.com"

Gebruik hetzelfde e-mailadres als je GitHub account — zo worden je commits correct gekoppeld aan je profiel.

3
Controleer je instellingen
Bash
git config --list

Je ziet een lijst van alle instellingen. Controleer of user.name en user.email correct staan.

⚘

GitHub account aanmaken

Gratis account op github.com — je online opslagplaats voor code

1
Ga naar github.com

Surf naar github.com en klik op Sign up (rechtsboven).

2
Vul je gegevens in

E-mailadres: gebruik je schoolmailadres
Wachtwoord: kies een sterk wachtwoord
Gebruikersnaam: kies iets professioneel, bv. jan-peeters-5adb. Je leerkracht herkent je hierdoor.

3
Bevestig je e-mailadres

GitHub stuurt een bevestigingsmail naar je schooladres. Klik op de link in die mail om je account te activeren.

4
Geef je gebruikersnaam door aan je leerkracht

Je leerkracht maakt een persoonlijke repository voor je aan. Je ontvangt dan een uitnodiging per mail om toegang te krijgen.

🔎 GitHub Student Developer Pack

Als student kun je gratis het GitHub Student Developer Pack aanvragen via education.github.com. Hiermee krijg je toegang tot gratis tools en diensten. Vraag dit aan met je schoolmailadres.

✓ Alles geïnstalleerd? Dan ben je klaar om te beginnen!

Je hebt Git geïnstalleerd, geconfigureerd en een GitHub account aangemaakt. Nu leer je in de cursus hoe je dit alles in de praktijk gebruikt.
