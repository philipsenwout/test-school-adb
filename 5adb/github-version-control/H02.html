H02 — GitHub Version Control 5ADB

Git configureren

ADB16De leerlingen gebruiken een versiebeheersysteem.
🎯 Leerdoelen
  • Je kan de terminal (Git Bash / Terminal) openen en basiscommando's uitvoeren
  • Je kan controleren of Git correct geïnstalleerd is via git --version
  • Je kan je naam en e-mailadres instellen via git config --global
  • Je kan navigeren in de terminal met cd, ls en pwd
ⓘ Vereiste voorbereiding

Heb je Git nog niet geïnstalleerd? Ga eerst naar de software installatiepagina en volg de stappen.

2.1 De terminal openen

De terminal is een tekstvenster waarmee je commando's geeft aan je computer. Git werkt via de terminal — je typt commando's en Git antwoordt met resultaten.

Windows — Git Bash

Git voor Windows installeert automatisch Git Bash, een Linux-achtige terminal. Je opent Git Bash op twee manieren:

  • Via het startmenu: zoek op "Git Bash"
  • Rechtermuisklik in een map → Open Git Bash here (handig!)
💡 Windows Terminal en PowerShell

Git werkt ook in Windows Terminal en PowerShell. Maar Git Bash heeft als voordeel dat alle commando's in dit boek (Linux-stijl) zonder aanpassing werken.

macOS — Terminal

Druk op Cmd+Spatie, typ "Terminal" en druk Enter. Of via Finder → Programma's → Hulpprogramma's → Terminal.

Linux

Druk op Ctrl+Alt+T of zoek "Terminal" in het toepassingenmenu.

2.2 Controleer of Git werkt

Open je terminal en typ:

Bash
git --version

Je zou zoiets moeten zien:

Bash
git version 2.47.0.windows.2
❌ Foutmelding: "git is not recognized"

Als je git: command not found of 'git' is not recognized ziet, is Git niet correct geïnstalleerd of niet toegevoegd aan het PATH. Ga terug naar de installatiepagina en controleer stap 2 (vink "Add to PATH" aan).

2.3 Je naam en e-mail instellen

Elke commit die je maakt, bevat jouw naam en e-mailadres. Zo weet de leerkracht van wie de commit is. Je stelt dit één keer in, en Git onthoudt het voor altijd.

Bash
git config --global user.name "Voornaam Naam"
git config --global user.email "jouw@schoolmail.be"
⚠ Gebruik je schoolmailadres

Het e-mailadres moet hetzelfde zijn als je GitHub account. Zo worden je commits correct gelinkt aan je profiel en kan de leerkracht je werk identificeren.

Controleer daarna of alles correct staat:

Bash
git config --list

Je ziet een lijst van alle instellingen. Zoek naar user.name en user.email.

2.4 Navigeren in de terminal

In de terminal navigeer je naar mappen (directories) met tekst-commando's. Dit zijn de drie commando's die je elke dag gebruikt:

pwd — Print Working Directory

Toont in welke map je je momenteel bevindt.

Bash
pwd
# Uitvoer: /c/Users/JanPeeters/Documents

ls — List

Toont de bestanden en mappen in de huidige map.

Bash
ls
# Toont: app-ontwikkeling/  README.md  huistaken/

cd — Change Directory

Navigeer naar een andere map.

Bash
cd Documents            # ga naar Documents
cd app-ontwikkeling     # ga naar een submap
cd ..                   # ga één map omhoog
cd ~                    # ga naar je home-map
💡 Tab-aanvulling

Druk op Tab terwijl je typt om mapnamen automatisch aan te vullen. Typ bv. cd Doc + Tab en de terminal vult aan naar cd Documents/. Zeer handig!

Oefeningen

ADB16 Oefening 1 — Git controleren en configureren

Voer de volgende stappen uit in de terminal:

  1. Open Git Bash (Windows) of Terminal (Mac/Linux)
  2. Voer git --version uit en noteer de versie die je ziet
  3. Stel je naam in: git config --global user.name "Jouw Naam"
  4. Stel je e-mail in: git config --global user.email "jouw@schoolmail.be"
  5. Controleer met git config --list — zijn naam en e-mail correct?
ADB16 Oefening 2 — Navigeren in de terminal

Navigeer zonder de muis naar een bepaalde map:

  1. Open een nieuwe terminal
  2. Gebruik pwd om te zien waar je bent
  3. Gebruik ls om de inhoud van de huidige map te zien
  4. Navigeer naar je Documenten map met cd
  5. Maak een nieuwe map aan: mkdir git-oefening
  6. Navigeer daarin: cd git-oefening
  7. Ga terug naar Documenten: cd ..
