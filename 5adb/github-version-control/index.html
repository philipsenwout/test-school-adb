GitHub Version Control

Leer Git gebruiken om je code bij te houden, in te dienen en samen te werken. Je kloont je persoonlijke repository, commit je werk en pusht het naar GitHub — elke dag.

📃 Hoe werkt dit vak?

GitHub Version Control is geen apart uur, maar loopt doorheen heel het jaar. Je gebruikt Git bij elk vak. Elke taak, toets en oefening die je maakt, sla je op in je persoonlijke GitHub-repository.
