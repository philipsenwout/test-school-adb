8.1 Waarom Python voor data?

SQL is ideaal voor het opvragen en aanpassen van gegevens in een database. Maar in de praktijk ontvang je data ook in andere formaten: Excel-bestanden, JSON-bestanden, CSV-bestanden …

Python is de meest gebruikte taal voor dataverwerking omdat:

  • het eenvoudig te lezen is
  • er veel handige bibliotheken voor data zijn (openpyxl, json, pandas …)
  • het automatisering mogelijk maakt: stel je voor dat je elke maand 50 Excel-bestanden moet verwerken
💡 Tip

Python installeer je via python.org. Zorg dat je ook openpyxl installeert:


pip install openpyxl

8.2 Excel met Python (openpyxl)

8.2.1 Een Excel-bestand openen


import openpyxl

# Werkmap openen
wb = openpyxl.load_workbook('cijfers.xlsx')

import openpyxl

# Eén cel opvragen
cel_a1 = ws['A1'].value
print(cel_a1)

for rij in ws.iter_rows(min_row=2, values_only=True):
    naam = rij[0]    # kolom A

8.2.3 Berekeningen op een kolom

Stel: je hebt een Excel-bestand cijfers.xlsx met twee kolommen:

  • Kolom A: naam van de leerling
  • Kolom B: score (getal van 0 tot 20)

import openpyxl

wb = openpyxl.load_workbook('cijfers.xlsx')
ws = wb.active

scores = []

# Alle scores verzamelen (sla de titelbalk over met min_row=2)
for rij in ws.iter_rows(min_row=2, values_only=True):
    naam = rij[0]
    score = rij[1]
    if score is not None:
        scores.append(score)
        print(f"{naam}: {score}/20")

# Berekeningen
totaal = sum(scores)
gemiddelde = totaal / len(scores)
hoogste = max(scores)
laagste = min(scores)

print(f"\nGemiddelde: {gemiddelde:.1f}")
print(f"Hoogste score: {hoogste}")
print(f"Laagste score: {laagste}")

8.2.4 Resultaat opslaan in een nieuw Excel-bestand


import openpyxl

# Nieuw werkmap aanmaken
wb_nieuw = openpyxl.Workbook()
ws_nieuw = wb_nieuw.active
ws_nieuw.title = "Resultaten"

# Kopteksten toevoegen
ws_nieuw['A1'] = 'Naam'
ws_nieuw['B1'] = 'Score'
ws_nieuw['C1'] = 'Geslaagd?'

# Data toevoegen
data = [
    ('Jan', 14),
    ('Sara', 8),
    ('Raf', 17),
]

for rij_nr, (naam, score) in enumerate(data, start=2):
    ws_nieuw.cell(row=rij_nr, column=1, value=naam)
    ws_nieuw.cell(row=rij_nr, column=2, value=score)
    ws_nieuw.cell(row=rij_nr, column=3, value='Ja' if score >= 10 else 'Nee')

# Opslaan
wb_nieuw.save('resultaten.xlsx')
print("Bestand opgeslagen!")

8.3 JSON met Python

8.3.1 Wat is JSON?

📘 Definitie

JSON (JavaScript Object Notation) is een tekstformaat voor het opslaan en uitwisselen van gestructureerde gegevens. Het lijkt op een Python-woordenboek (dictionary).

JSON wordt veel gebruikt bij API's (koppeling tussen applicaties), configuratiebestanden en webapplicaties. Je zal het ook tegenkomen in App-ontwikkeling.

Voorbeeld van een JSON-bestand (producten.json):


{
  "producten": [
    {"id": 1, "naam": "Laptop", "prijs": 999.99},
    {"id": 2, "naam": "Muis", "prijs": 24.95},
    {"id": 3, "naam": "Toetsenbord", "prijs": 44.50}
  ]
}

8.3.2 JSON inlezen


import json

# JSON-bestand openen en inlezen
with open('producten.json', 'r', encoding='utf-8') as bestand:
    data = json.load(bestand)

# Data raadplegen
print(data['producten'])           # de volledige lijst
print(data['producten'][0])        # eerste product
print(data['producten'][0]['naam']) # naam van eerste product

# Door alle producten lopen
for product in data['producten']:
    print(f"{product['naam']}: €{product['prijs']}")

8.3.3 JSON wegschrijven


import json

# Data die we willen opslaan
leerlingen = [
    {"naam": "Jan Peeters", "klas": "5ADB", "score": 14},
    {"naam": "Sara Claes", "klas": "5ADB", "score": 17},
    {"naam": "Raf Dubois", "klas": "5ADB", "score": 9},
]

data = {"leerlingen": leerlingen}

# Wegschrijven naar JSON-bestand
with open('leerlingen.json', 'w', encoding='utf-8') as bestand:
    json.dump(data, bestand, indent=4, ensure_ascii=False)

print("JSON-bestand opgeslagen!")
💡 Tip

indent=4 zorgt voor mooie opmaak met inspringen. ensure_ascii=False laat speciale tekens (é, à, ë …) correct opslaan.

Voorbeelden

🟢 Voorbeeld

Je krijgt een Excel-bestand punten_5adb.xlsx met de kolommen: Naam, Toets1, Toets2, Toets3.


import openpyxl

wb = openpyxl.load_workbook('punten_5adb.xlsx')
ws = wb.active

for rij in ws.iter_rows(min_row=2, values_only=True):
naam, t1, t2, t3 = rij
if naam and t1 and t2 and t3:
gemiddelde = (t1 + t2 + t3) / 3
print(f"{naam}: gemiddelde = {gemiddelde:.1f}/20")
🟢 Voorbeeld

import json

# Inlezen
with open('producten.json', 'r') as f:
data = json.load(f)

# Nieuw product toevoegen
nieuw_product = {"id": 4, "naam": "Monitor", "prijs": 299.00}
data['producten'].append(nieuw_product)

# Wegschrijven
with open('producten.json', 'w') as f:
json.dump(data, f, indent=4)

print("Product toegevoegd!")

Samenvatting

📋 Samenvatting
  • Python is nuttig om data te verwerken in Excel- en JSON-formaten.
  • openpyxl: Excel-bestanden openen (load_workbook), rijen doorlopen (iter_rows), cellen lezen en schrijven, opslaan (save).
