9.1 Waarom visualiseren?

Een tabel met 100 rijen is moeilijk te lezen. Een grafiek toont direct:

  • Welke maand de meeste verkopen had
  • Hoe scores over de klas verdeeld zijn
  • Welk product het populairst is
💡 Tip

Installeer matplotlib via de terminal:


pip install matplotlib

9.2 Lijndiagram

Een lijndiagram is ideaal om een evolutie over de tijd te tonen.


import matplotlib.pyplot as plt

# Data
maanden = ['Jan', 'Feb', 'Mar', 'Apr', 'Mei', 'Jun']
verkopen = [120, 145, 98, 160, 175, 130]

# Grafiek aanmaken
plt.plot(maanden, verkopen, marker='o', color='blue')

# Labels en titel
plt.title('Verkopen per maand')
plt.xlabel('Maand')
plt.ylabel('Aantal verkopen')

# Tonen
plt.show()

!Lijndiagram voorbeeld

Figuur 9.1 — Lijndiagram van verkopen per maand

💡 Tip

marker='o' zet een rondje op elk datapunt. Andere opties: 's' (vierkant), '^' (driehoek), '*' (ster).

9.3 Staafdiagram

Een staafdiagram toont een vergelijking tussen categorieën.


import matplotlib.pyplot as plt

# Data
vakken = ['Netwerken', 'App-ontwikkeling', 'Databeheer', 'E-STEM']
gemiddelden = [12.5, 14.2, 11.8, 13.1]

# Grafiek
plt.bar(vakken, gemiddelden, color='steelblue')

# Labels
plt.title('Gemiddelde score per vak — 5ADB')
plt.xlabel('Vak')
plt.ylabel('Gemiddelde score (/20)')

# Y-as van 0 tot 20
plt.ylim(0, 20)

plt.show()

9.4 Cirkeldiagram

Een cirkeldiagram toont verhoudingen (hoe groot is elk stuk van het geheel).


import matplotlib.pyplot as plt

# Data
categorieën = ['Elektronica', 'Kleding', 'Voeding', 'Sport']
aandelen = [35, 25, 20, 20]

# Grafiek
plt.pie(aandelen, labels=categorieën, autopct='%1.1f%%', startangle=90)

plt.title('Verkopen per categorie')

plt.show()
💡 Tip

autopct='%1.1f%%' toont het percentage op elk taartpunt. startangle=90 begint het eerste stuk bovenaan.

9.5 Grafiek opslaan als bestand


import matplotlib.pyplot as plt

maanden = ['Jan', 'Feb', 'Mar']
verkopen = [120, 145, 98]

plt.bar(maanden, verkopen, color='green')
plt.title('Verkopen Q1')

# Opslaan VOOR plt.show()
plt.savefig('verkopen_q1.png', dpi=150)

plt.show()
print("Grafiek opgeslagen!")

9.6 Meerdere grafieken per figuur


import matplotlib.pyplot as plt

# Twee grafieken naast elkaar
fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(10, 4))

# Eerste grafiek
ax1.bar(['Jan', 'Feb', 'Mar'], [120, 145, 98], color='steelblue')
ax1.set_title('Verkopen Q1')

# Tweede grafiek
ax2.plot(['Jan', 'Feb', 'Mar'], [12.5, 13.2, 11.8], marker='o', color='red')
ax2.set_title('Gemiddelde score Q1')

plt.tight_layout()
plt.show()

Voorbeelden

🟢 Voorbeeld

import matplotlib.pyplot as plt

namen = ['Jan', 'Sara', 'Raf', 'Lena', 'Tom']
scores = [14, 17, 9, 12, 15]

kleuren = ['green' if s >= 10 else 'red' for s in scores]

plt.bar(namen, scores, color=kleuren)
plt.axhline(y=10, color='black', linestyle='--', label='Grens geslaagd')
plt.title('Scores toets 1 — 5ADB')
plt.ylabel('Score (/20)')
plt.ylim(0, 20)
plt.legend()
plt.savefig('scores_toets1.png')
plt.show()
🟢 Voorbeeld

import openpyxl
import matplotlib.pyplot as plt

# Excel inlezen
wb = openpyxl.load_workbook('maandrapport.xlsx')
ws = wb.active

maanden = []
verkopen = []

for rij in ws.iter_rows(min_row=2, values_only=True):
if rij[0] and rij[1]:
maanden.append(rij[0])
verkopen.append(rij[1])

# Grafiek
plt.plot(maanden, verkopen, marker='o')
plt.title('Maandrapport')
plt.xlabel('Maand')
plt.ylabel('Verkopen')
plt.savefig('maandrapport.png')
plt.show()

Samenvatting

📋 Samenvatting
  • Datavisualisatie maakt patronen in data zichtbaar die in een tabel moeilijk te zien zijn.
  • Lijndiagram (plt.plot): evolutie over tijd.
  • Staafdiagram (plt.bar): vergelijking tussen categorieën.
  • Cirkeldiagram (plt.pie): verhoudingen.
  • Altijd toevoegen: titel (plt.title), aslabels (plt.xlabel, plt.ylabel).
  • Grafiek opslaan: plt.savefig('naam.png') — vóór plt.show()!
