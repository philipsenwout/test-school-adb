4.1 SSH — verbinding op afstand

📘 Definitie

SSH is een versleuteld protocol dat een veilige verbinding biedt tussen twee computers via het netwerk. Je kan via SSH commando's uitvoeren op de Raspberry Pi zonder monitor of keyboard op de Pi.

💡 Tip

SSH werkt via TCP/IP op poort 22. Dit is dezelfde poort- en protocollogica die je in het vak Netwerken leert.

!SSH verbinding

Figuur 4.1 — SSH verbindt laptop en Raspberry Pi via het netwerk

SSH activeren op de Pi

SSH is ingebouwd in Raspberry Pi OS. Zorg dat het geactiveerd is:

  1. Open een terminal op de Pi
  2. Voer uit: sudo raspi-config
  3. Kies: Interface Options → SSH → Enable

SSH-verbinding maken

Vanuit Windows (PowerShell):


ssh pi@192.168.1.105
# of via hostname:
ssh pi@pi-jan.local

Stappen:

  1. Open PowerShell of een terminal op je laptop
  2. Typ ssh pi@[ip-adres-van-pi]
  3. Eerste keer: bevestig met yes
  4. Voer het wachtwoord in (niet zichtbaar bij typen)

Je bent nu ingelogd op de Raspberry Pi!

4.2 IP-adres van de Pi vinden


# Op de Pi zelf:
hostname -I

# Meer details:
ip addr show

Of via Windows:


# Ping de Pi om te zien of ze bereikbaar is
ping pi-jan.local
💡 Tip

In de router-instellingen (typ 192.168.1.1 in je browser) zie je een lijst van alle verbonden apparaten met hun IP-adressen. Handig om het adres van de Pi te vinden als je het niet weet.

4.3 Bestanden kopiëren via SCP

SCP (Secure Copy Protocol) kopieert bestanden via SSH.


# Bestand van je laptop naar de Pi sturen:
scp mijnscript.py pi@192.168.1.105:/home/pi/

# Bestand van de Pi naar je laptop halen:
scp pi@192.168.1.105:/home/pi/resultaten.csv ./

# Een volledige map kopiëren:
scp -r projectmap/ pi@192.168.1.105:/home/pi/

4.4 Netwerkconfiguratie bekijken


# IP-adres en netwerkinformatie
ip addr show

# Standaardgateway (= je router)
ip route show

# DNS-configuratie
cat /etc/resolv.conf

# Netwerkverbindingen testen
ping 8.8.8.8          # ping naar Google DNS
ping google.com       # test DNS + connectiviteit

# Traceroute (hoe pakketten reizen)
traceroute google.com
💡 Tip

ip addr show geeft het IP-adres en subnetmasker. ip route show toont de standaardgateway. Dit zijn dezelfde concepten die je in het vak Netwerken leert bij IPv4 en subnetting.

4.5 Python-scripts uitvoeren via SSH

Via SSH kan je Python-scripts schrijven en uitvoeren zonder monitor op de Pi.

Workflow:


# 1. Verbinden via SSH
ssh pi@192.168.1.105

# 2. Script aanmaken
nano hallo.py

# 3. Script uitvoeren
python3 hallo.py

Voorbeeld — systeminformatie script:


import subprocess

# Voer een terminalcommando uit vanuit Python
resultaat = subprocess.run(['hostname', '-I'], capture_output=True, text=True)
ip = resultaat.stdout.strip()

print(f"IP-adres van de Pi: {ip}")
print("Dit script draait op de Raspberry Pi!")

Script naar Pi sturen en uitvoeren:


# Op je laptop: bestand schrijven
# Op je laptop: scp gebruiken om het te sturen
scp info.py pi@192.168.1.105:/home/pi/

# SSH inloggen en uitvoeren
ssh pi@192.168.1.105
python3 info.py

🟢 Voorbeeld

import subprocess
from datetime import datetime

def run_cmd(cmd):
resultaat = subprocess.run(cmd, capture_output=True, text=True, shell=True)
return resultaat.stdout.strip()

print("=== Raspberry Pi Systeminformatie ===")
print(f"Datum/tijd: {datetime.now().strftime('%Y-%m-%d %H:%M')}")
print(f"Hostname:   {run_cmd('hostname')}")
print(f"IP-adres:   {run_cmd('hostname -I')}")
print(f"OS:         {run_cmd('uname -r')}")
print("=====================================")
