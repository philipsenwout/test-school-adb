3.1 Python starten op de Pi

Python 3 is voorgeïnstalleerd op Raspberry Pi OS.

Interactieve modus (snel testen):


python3
>>> print("Hallo wereld!")
Hallo wereld!
>>> 3 + 5
8
>>> exit()

Script schrijven en uitvoeren:


nano mijnscript.py    # script schrijven
python3 mijnscript.py # script uitvoeren

3.2 Variabelen en datatypes


# Variabelen
naam = "Sara"           # str (tekst)
leeftijd = 17           # int (geheel getal)
score = 14.5            # float (decimaal getal)
geslaagd = True         # bool (True of False)

print(naam)
print(leeftijd)
print(type(leeftijd))   # <class 'int'>

Omrekenen tussen types:


getal_tekst = "42"

kommagetal = float("3.14") # str → float
tekst = str(100)           # int → str

3.3 Invoer van de gebruiker


naam = input("Wat is je naam? ")
print("Hallo, " + naam + "!")

leeftijd = int(input("Hoe oud ben je? "))
print("Je bent", leeftijd, "jaar oud.")
⚠️ Let op

️ Let op

input() geeft altijd een tekst (str) terug. Als je ermee wil rekenen, moet je het omzetten met int() of float().

3.4 Condities: if / elif / else


score = int(input("Geef je score: "))

if score >= 10:
    print("Geslaagd!")
elif score >= 8:
    print("Bijna — nog een kleine inspanning.")
else:
    print("Niet geslaagd.")

Vergelijkingsoperatoren:

OperatorBetekenis
==gelijk aan
!=niet gelijk aan
>groter dan
<kleiner dan
>=groter dan of gelijk aan
<=kleiner dan of gelijk aan

# Meerdere condities combineren
if score >= 10 and score <= 20:
    print("Geldige score en geslaagd!")

if naam == "admin" or naam == "beheerder":
    print("Welkom, beheerder!")

3.5 Lussen

For-lus


# Tabel van 3
for i in range(1, 11):
    print(f"3 × {i} = {3 * i}")

# Over een lijst lopen
vakken = ["Netwerken", "Python", "Databeheer"]
for vak in vakken:
    print("Vak:", vak)

While-lus


# Blijf vragen tot het juiste wachtwoord
wachtwoord = ""
while wachtwoord != "geheim":
    wachtwoord = input("Wachtwoord: ")
    if wachtwoord != "geheim":
        print("Fout wachtwoord, probeer opnieuw.")

print("Toegang verleend!")
💡 Tip

Gebruik een for-lus als je weet hoeveel keer je wil herhalen.

Gebruik een while-lus als je wil herhalen zolang een voorwaarde geldt.

3.6 Functies


def begroet(naam):
    print("Hallo, " + naam + "!")

begroet("Sara")
begroet("Jan")

Functie met return-waarde:


def bereken_gemiddelde(score1, score2, score3):
    totaal = score1 + score2 + score3
    gemiddelde = totaal / 3
    return gemiddelde

resultaat = bereken_gemiddelde(14, 12, 16)
print(f"Gemiddelde: {resultaat:.1f}")

Waarom functies?

  • Code hergebruiken zonder te kopiëren
  • Programma overzichtelijk houden
  • Elke functie doet één ding

Voorbeelden

🟢 Voorbeeld

def reken(a, b, operator):
if operator == '+':
return a + b
elif operator == '-':
return a - b
elif operator == '*':
return a * b
elif operator == '/':
if b != 0:
return a / b
else:
return "Kan niet delen door nul!"
else:
return "Onbekende operator"

a = float(input("Eerste getal: "))
operator = input("Operator (+, -, *, /): ")
b = float(input("Tweede getal: "))

print("Resultaat:", reken(a, b, operator))
🟢 Voorbeeld

def druk_tafel(n):
print(f"\nTafel van {n}:")
for i in range(1, 11):
print(f"  {n} × {i} = {n * i}")

for tafel in [2, 3, 5, 10]:
druk_tafel(tafel)

Samenvatting

📋 Samenvatting
  • Python schrijf je in een .py-bestand, uitvoeren via python3 bestand.py.
  • Datatypes: str (tekst), int (geheel), float (decimaal), bool (waar/niet waar).
  • input() geeft altijd een string terug — gebruik int() of float() om te rekenen.
  • if/elif/else voor beslissingen.
  • for voor een bekend aantal herhalingen; while zolang een voorwaarde geldt.
  • def om een functie te definiëren; functies maken code herbruikbaar en overzichtelijk.
