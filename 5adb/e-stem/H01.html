1.1 Wat is STEM?

📘 Definitie

STEM staat voor Science (wetenschap), Technology (technologie), Engineering (ingenieurskunde) en Mathematics (wiskunde). Het is een aanpak waarbij deze vier domeinen samen worden ingezet om problemen op te lossen.

In de richting Data- en Applicatiebeheer werk je dagelijks met technologie: applicaties bouwen, netwerken configureren, databases beheren. E-STEM legt de verbinding tussen deze digitale wereld en echte hardware.

EngineeringProblemen oplossen met een stappenplan
MathematicsBerekeningen in Python-scripts

1.2 De Raspberry Pi

📘 Definitie

De Raspberry Pi is een kleine, goedkope computer ter grootte van een creditcard. Hij draait een volledig besturingssysteem (Linux) en kan worden gebruikt voor programmeren, netwerken, webservers en nog veel meer.

!Raspberry Pi hardware overzicht

Figuur 1.1 — De onderdelen van een Raspberry Pi

Kenmerken van de Raspberry Pi 4:

OnderdeelBeschrijving
ProcessorQuad-core ARM, 1,5 GHz
RAM2 GB (of meer)
OpslagMicroSD-kaart
AansluitingenUSB, HDMI, Ethernet, WiFi, Bluetooth
GPIO-pinnen40 pinnen voor hardware-aansluitingen
BesturingssysteemRaspberry Pi OS (Linux)

Waarvoor gebruiken we de Raspberry Pi in dit vak?

  • Linux leren via de terminal
  • Python-scripts schrijven en uitvoeren
  • De Pi als kleine server of netwerktoestel gebruiken
  • Verbinding maken via SSH (op afstand beheren)

1.3 Raspberry Pi OS installeren

Raspberry Pi OS is een Linux-distributie op basis van Debian. Je installeert het op een microSD-kaart via de Raspberry Pi Imager.

Stappen:

  1. Download Raspberry Pi Imager via raspberrypi.com/software
  2. Steek een microSD-kaart in je computer
  3. Open de Imager → kies Raspberry Pi OS (64-bit)
  4. Kies de microSD als bestemming
  5. Klik WRITE en wacht tot het klaar is
  6. Steek de SD-kaart in de Pi en sluit stroom aan

Eerste opstart:

  • Sluit monitor, toetsenbord en muis aan (of gebruik SSH)
  • De Pi start op in het Linux-bureaublad
  • Stel taal, tijdzone en wachtwoord in
⚠️ Let op

️ Let op

Kies altijd een sterk wachtwoord bij de eerste setup. De Pi is een volwaardige computer — een zwak wachtwoord is een beveiligingsrisico.

1.4 Het bureaublad verkennen

Het Raspberry Pi OS-bureaublad lijkt op Windows of macOS:

  • Menubalk bovenaan — toepassingen starten, WiFi-instellingen
  • Bestandsbeheer — door mappen navigeren
  • Terminal — commandovenster (het belangrijkste hulpmiddel in dit vak!)
  • Chromium — webbrowser
💡 Tip

De terminal open je via Menu → Accessoires → Terminal, of met de sneltoets Ctrl + Alt + T. In de terminal typ je commando's om de Pi te beheren.

1.5 Het ontwerpproces

In E-STEM werk je met een gestructureerd ontwerpproces om problemen op te lossen.

StapNaamWat je doet
1ProbleemstellingFormuleer het probleem helder: wat moet de oplossing kunnen?
2BrainstormGenereer zoveel mogelijk ideeën zonder oordeel
3SelectieBeoordeel ideeën op criteria en kies de beste oplossing
4Ontwerp & BouwSchrijf de code, maak het script
5Test & EvalueerWerkt het? Wat kan beter?
💡 Tip

Een goede probleemstelling begint met: "Schrijf een Python-script dat … zodat …". Wees specifiek over het doel en de randvoorwaarden.

Voorbeelden

🟢 Voorbeeld

Vaag (niet goed): "Maak iets met Python."

Goed: "Schrijf een Python-script dat de gebruiker naar zijn naam vraagt en hem begroet met 'Goeiemorgen, [naam]!'. Het script herhaalt dit totdat de gebruiker 'stop' intypt."

De goede formulering bevat: doel (begroeten), invoer (naam), randvoorwaarde (herhalen tot 'stop').
