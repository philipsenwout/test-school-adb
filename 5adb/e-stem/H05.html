5.1 Tekstbestanden lezen en schrijven

Bestand schrijven


# Bestand aanmaken en schrijven
with open('notities.txt', 'w') as bestand:
    bestand.write("Dit is mijn eerste notitie.\n")
    bestand.write("En dit is de tweede regel.\n")

print("Bestand aangemaakt!")
💡 Tip

Het with-blok sluit het bestand automatisch als het klaar is. Gebruik altijd with open() in plaats van apart open() en close().

Modi:

  • 'w' — schrijven (overschrijft bestaand bestand)
  • 'a' — aanvullen (voegt toe aan bestaand bestand)
  • 'r' — lezen (standaard)

Bestand lezen


# Volledige inhoud lezen
with open('notities.txt', 'r') as bestand:
    inhoud = bestand.read()
    print(inhoud)

# Regel per regel lezen
with open('notities.txt', 'r') as bestand:
    for regel in bestand:
        print(regel.strip())  # .strip() verwijdert spaties en \n

5.2 CSV-bestanden verwerken

📘 Definitie

CSV (Comma-Separated Values) is een eenvoudig tekstformaat voor data in tabelvorm. Elke rij is een lijn, de kolommen zijn gescheiden door komma's.

Voorbeeld van een CSV-bestand scores.csv:


naam,score1,score2,score3
Jan,14,12,15
Sara,17,16,18
Raf,8,10,9

CSV inlezen


import csv

with open('scores.csv', 'r') as bestand:
    lezer = csv.reader(bestand)
    next(lezer)  # sla de eerste rij (koptekst) over

    for rij in lezer:
        naam = rij[0]
        score1 = int(rij[1])
        score2 = int(rij[2])
        score3 = int(rij[3])
        print(f"{naam}: {score1}, {score2}, {score3}")

CSV lezen als woordenboek


import csv

with open('scores.csv', 'r') as bestand:
    lezer = csv.DictReader(bestand)  # gebruikt koptekst als kolomnaam

    for rij in lezer:
        naam = rij['naam']
        gemiddelde = (int(rij['score1']) + int(rij['score2']) + int(rij['score3'])) / 3
        print(f"{naam}: gemiddelde = {gemiddelde:.1f}")

CSV schrijven


import csv

resultaten = [
    ['naam', 'gemiddelde', 'geslaagd'],
    ['Jan', 13.7, 'Ja'],
    ['Sara', 17.0, 'Ja'],
    ['Raf', 9.0, 'Nee'],
]

with open('resultaten.csv', 'w', newline='') as bestand:
    schrijver = csv.writer(bestand)
    schrijver.writerows(resultaten)

print("CSV opgeslagen!")

5.3 Berekeningen op data


import csv

scores = []

with open('scores.csv', 'r') as bestand:
    lezer = csv.DictReader(bestand)
    for rij in lezer:
        gem = (int(rij['score1']) + int(rij['score2']) + int(rij['score3'])) / 3
        scores.append({'naam': rij['naam'], 'gemiddelde': gem})

# Statistieken
gemiddelden = [s['gemiddelde'] for s in scores]
print(f"Hoogste gemiddelde: {max(gemiddelden):.1f}")
print(f"Laagste gemiddelde: {min(gemiddelden):.1f}")
print(f"Klasgemiddelde: {sum(gemiddelden)/len(gemiddelden):.1f}")

Samenvatting

📋 Samenvatting
  • with open(bestand, modus): bestand openen en automatisch sluiten. Modi: 'r' (lezen), 'w' (schrijven), 'a' (aanvullen).
  • CSV: data in tabelvorm als platte tekst. Python leest/schrijft CSV via de csv-module.
  • csv.DictReader: leest CSV met kolomnamen als sleutels — overzichtelijker dan index.
  • csv.writer: schrijft rijen naar een CSV-bestand.
  • Combineer bestanden lezen/schrijven met berekeningen voor nuttige dataverwerkingsscripts.
