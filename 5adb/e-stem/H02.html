2.1 Wat is een terminal?

📘 Definitie

De terminal (ook: command line of CLI) is een tekstvenster waarin je commando's typt om de computer te besturen. In plaats van op pictogrammen te klikken, typ je instructies.

Waarom de terminal gebruiken?

  • Veel sneller dan klikken voor herhaalde taken
  • Noodzakelijk voor serverbeheer (servers hebben vaak geen grafische interface)
  • Basis van Linux- en netwerksysteembeheer

Open de terminal: Menu → Accessoires → Terminal of Ctrl + Alt + T

2.2 Navigatiecommando's

pwd — huidige map tonen


pwd

Uitvoer: /home/pi — je staat in de hoofmap van de gebruiker "pi"

ls — inhoud van een map tonen


ls          # eenvoudige lijst
ls -l       # gedetailleerde lijst (met rechten, datum, grootte)
ls -la      # ook verborgen bestanden tonen (beginnen met .)

cd — van map wisselen


cd Desktop          # ga naar de map Desktop
cd ..               # ga één map omhoog
cd /                # ga naar de rootmap
cd ~                # ga naar je hoofmap (/home/pi)
cd /home/pi/Desktop # absoluut pad (vanuit elke map)
💡 Tip

Druk op Tab na de eerste letters van een mapnaam om automatisch aan te vullen. Bv.: cd Des + Tab → cd Desktop/. Dit vermijdt typfouten.

2.3 Bestanden en mappen beheren

mkdir — map aanmaken


mkdir mijnmap
mkdir projecten/estem    # map aanmaken in een submap

touch — leeg bestand aanmaken


touch notities.txt
touch script.py

cp — kopiëren


cp notities.txt backup.txt           # bestand kopiëren
cp -r mijnmap mapkopie               # map kopiëren (-r = recursief)

mv — verplaatsen of hernoemen


mv notities.txt Desktop/notities.txt   # verplaatsen
mv oud_naam.txt nieuw_naam.txt         # hernoemen

rm — verwijderen


rm notities.txt          # bestand verwijderen
rm -r mijnmap            # map verwijderen (inclusief inhoud)
⚠️ Let op

️ Let op

rm verwijdert bestanden permanent — geen prullenbak op Linux! Dubbelcheck altijd voor je rm uitvoert.

2.4 Bestandsinhoud bekijken en bewerken

cat — inhoud tonen


cat notities.txt

nano — tekstbestand bewerken


nano notities.txt

In nano:

  • Typ je tekst gewoon
  • Ctrl + S — opslaan
  • Ctrl + X — afsluiten
  • Ctrl + K — regel knippen
  • Ctrl + U — plakken

2.5 Systeem- en netwerkcommando's


uname -a          # Linux-versie tonen
hostname          # naam van de computer
hostname -I       # IP-adres van de Pi
df -h             # schijfruimte tonen
free -h           # geheugengebruik tonen
top               # processen bekijken (Ctrl+C om te stoppen)

2.6 Software installeren met apt


sudo apt update              # lijst van beschikbare software bijwerken
sudo apt install [pakket]    # software installeren
sudo apt upgrade             # alle geïnstalleerde software bijwerken

Voorbeeld — Python pip installeren:


sudo apt update
sudo apt install python3-pip
💡 Tip

sudo staat voor "superuser do" — het voert een commando uit met beheerdersrechten. Heb je een foutmelding "permission denied"? Probeer sudo ervoor.

Huistaak H02

Mappenstructuur aanmaken

Maak op de Raspberry Pi (of thuis op je eigen Linux/Mac/Windows terminal) de volgende structuur aan:


schooljaar/
├── netwerken/
│   └── notities.txt
├── app-ontwikkeling/
│   └── notities.txt
├── databeheer/
│   └── notities.txt
└── e-stem/
    └── notities.txt

Voeg in elk notities.txt-bestand een korte beschrijving toe van wat je over dat vak al geleerd hebt (via nano).

Maak een screenshot van het eindresultaat (gebruik ls -R schooljaar/ om de volledige structuur te tonen).

Samenvatting

📋 Samenvatting
  • De terminal is het hart van Linux-beheer: commando's typen is sneller en krachtiger dan klikken.
  • Navigeren: pwd (huidige map), ls (inhoud), cd (wisselen van map).
  • Beheren: mkdir (map aanmaken), touch (bestand aanmaken), cp (kopiëren), mv (verplaatsen), rm (verwijderen).
  • Bewerken: nano voor tekstbestanden, cat voor snel bekijken.
  • Installeren: sudo apt update + sudo apt install.
  • Gebruik Tab voor auto-aanvulling — minder typewerk, minder fouten.
