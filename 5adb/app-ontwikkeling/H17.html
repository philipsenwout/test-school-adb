Variabelen

ADB09De leerlingen ontwikkelen oplossingen voor concrete problemen.
🎯 Leerdoelen
  • Je kan uitleggen wat een variabele is en waarvoor je het gebruikt
  • Je kent het verschil tussen let en const
  • Je kan variabelen correct declareren, initialiseren en een nieuwe waarde geven
  • Je kan de naamgevingsregels voor variabelen toepassen (camelCase)
  • Je kan console.log() gebruiken om waarden te tonen in de browser-console
  • Je kan "use strict" begrijpen en toepassen
🚀 Welkom bij JavaScript!

Vanaf dit hoofdstuk starten we met JavaScript — de programmeertaal van het web. Met JavaScript maak je je pagina's interactief: knoppen die reageren, berekeningen uitvoeren, data opslaan. CSS maakt het mooi, JavaScript maakt het slim.

1. Wat is een variabele?

Een variabele is een benoemd opslagplaatsje in het geheugen van de computer. Je kan er een waarde in bewaren, die waarde opvragen en (soms) later aanpassen.

Stel je voor: een variabele is een doosje met een label op.

  • Het label = de naam van de variabele
  • De inhoud = de waarde
JavaScript
let leeftijd = 16;

Dit maakt een doosje met label leeftijd en stopt er de waarde 16 in.

2. let en const

2.1 let — een aanpasbare variabele

JavaScript
let score = 0;
score = 10;  // de waarde aanpassen mag
score = 25;  // opnieuw aanpassen mag ook

Gebruik let als de waarde later in je programma kan veranderen (bijv. een teller, een score, een gebruikersinvoer).

2.2 const — een constante waarde

JavaScript
const maxPunten = 100;
// maxPunten = 200; // FOUT — dit mag niet!

Gebruik const als de waarde nooit mag veranderen (bijv. een maximumscore, een kleurcode, een vaste tekst).

💡 Vuistregel

Gebruik altijd const tenzij je weet dat je de waarde later moet aanpassen — gebruik dan let.

2.3 var — de oude manier (vermijden)

Je ziet in oudere code ook var. Dat werkt nog, maar heeft verwarrende regels rond scope. Gebruik nooit var in nieuwe code — kies altijd let of const.

3. Een variabele declareren en initialiseren

JavaScript
// Declareren én initialiseren in één stap
let naam = "Jana";

// Declareren zonder waarde (waarde = undefined)
let resultaat;

// Later een waarde toekennen
resultaat = 42;
JavaScript
let x = 10;
x = 11;     // correct — geen let
// let x = 12; // fout — let mag maar één keer per variabele

4. Naamgevingsregels

4.1 Wat mag?

  • Letters, cijfers, _ en $
  • Mag niet beginnen met een cijfer
  • Hoofdlettergevoelig: naam en Naam zijn twee verschillende variabelen
JavaScript
let leeftijd = 16;     // correct
let maxPunten = 100;   // correct — camelCase
// let 1ste = "eerste"; // FOUT — begint met cijfer
// let mijn naam = "x"; // FOUT — spatie in naam

4.2 camelCase

JavaScript-ontwikkelaars gebruiken standaard camelCase: het eerste woord in kleine letters, elk volgend woord begint met een hoofdletter.

JavaScript
let voornaam = "Jana";
let achternaam = "Peeters";
let geboortejaar = 2009;
let isAangemeld = false;
let maximaleSnelheid = 120;

4.3 Betekenisvolle namen

JavaScript
// niet duidelijk
let x = 16;
let y = "Jana";

// duidelijk
let leeftijd = 16;
let naam = "Jana";

5. console.log() — jouw beste vriend

console.log() toont een waarde in de developer console van de browser. Dit is de manier om te controleren of je code correct werkt.

JavaScript
let naam = "Jana";
let leeftijd = 16;

console.log(naam);              // toont: Jana
console.log(leeftijd);          // toont: 16
console.log("Naam:", naam);     // toont: Naam: Jana
console.log(naam, leeftijd);    // toont: Jana 16

Console openen in de browser: druk op F12 (of Ctrl+Shift+I) → tabblad "Console".

6. "use strict"

JavaScript
"use strict";

let x = 10;

"use strict" zet JavaScript in de strikte modus. Dit:

  • Voorkomt stilzwijgende fouten (bv. een variabele gebruiken zonder let/const geeft een fout)
  • Maakt je code veiliger en voorspelbaarder

Zet "use strict" altijd bovenaan je JavaScript-bestand.

7. JavaScript koppelen aan HTML

7.1 Extern bestand (aanbevolen)

HTML
<!-- index.html -->
<body>
  <h1>Mijn pagina</h1>

  <!-- Script altijd onderaan body, net voor </body> -->
  <script src="script.js"></script>
</body>
JavaScript
// script.js
"use strict";
let naam = "Jana";
console.log("Hallo,", naam);

7.2 Inline in HTML (alleen voor kleine tests)

HTML
<script>
  let naam = "Jana";
  console.log(naam);
</script>
💡 Waarom script onderaan body?

De browser leest HTML van boven naar beneden. Als het script bovenaan staat, probeert het al elementen te manipuleren die nog niet bestaan. Onderaan <body> staat de hele HTML al klaar.

Oefeningen

ADB09 Oefening 1 — Variabelenpagina

Maak een bestand variabelen.html met een <script>-tag (of extern script.js):

  1. Declareer met const: je naam, je geboortejaar, je favoriete kleur
  2. Declareer met let: een teller die begint op 0, een score die begint op 0
  3. Pas de teller aan naar 1, daarna naar 2
  4. Pas de score aan naar 50, daarna naar 75
  5. Gebruik console.log() om alle variabelen te tonen
  6. Probeer een const-variabele te wijzigen — wat zegt de console?

Huistaak

📋 Huistaak — Variabelen

Opdracht: Mijn profiel in variabelen

  1. Deel 1 — const-variabelen (4 punten): Declareer minstens 5 constante variabelen over jezelf: naam, geboortejaar, favoriete vak, woonplaats en één extra naar keuze. Gebruik beschrijvende camelCase namen.
  2. Deel 2 — let-variabelen (3 punten): Declareer 3 variabelen met let die je daarna aanpast: een teller (verhoog 3 keer), een score (verander 2 keer), een status (stel in op "aangemeld").
  3. Deel 3 — console.log (3 punten): Toon alle variabelen via console.log() met een duidelijk label voor elke waarde. Voeg "use strict" toe bovenaan je script.
