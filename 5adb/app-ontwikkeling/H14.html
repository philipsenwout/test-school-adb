H14 — App-ontwikkeling 5ADB

Float en Positioning

ADB10De leerlingen werken een applicatie uit binnen de opgelegde standaardsoftware of het IT-platform.
📌 Leerdoelen
  • Uitleggen wat float doet en wanneer het nog zinvol is
  • float: left en float: right toepassen voor tekst rondom een afbeelding
  • clear gebruiken om het float-effect te stoppen
  • De vijf positioneringsmodi begrijpen: static, relative, absolute, fixed, sticky
  • z-index gebruiken om elementen voor of achter elkaar te plaatsen
  • Correct inschatten wanneer je float/position gebruikt en wanneer je beter kiest voor Flexbox of Grid
"Float and position are for special cases, not for general layout. For that, use Flex or Grid."
— Kevin Powell

1. Historische context — waarom leren we dit nog?

Tot ongeveer 2015 was float de standaard manier om lay-outs te bouwen. Je vindt nog veel websites, tutorials en code online die van float gebruikmaakten. Het is dus nuttig om te begrijpen wat het doet.

Maar: voor algemene lay-out gebruik je vandaag Flexbox of Grid. Float en position zijn gereserveerd voor specifieke doeleinden.

💡 Gebruik float voor:
  • Tekst die om een afbeelding heen loopt
  • Een uitgelicht citaat (blockquote) naast tekst
💡 Gebruik position voor:
  • Een element op een nauwkeurige plek binnen een container zetten
  • Een sticky navigatiebalk die bovenaan blijft
  • Een overlay of badge over een afbeelding

2. Float

2.1 Tekst laten vloeien rond een afbeelding

Het meest klassieke gebruik van float: een afbeelding staat links en de tekst vloeit ernaast.

HTML
<p>
    <img class="links" src="paraplu.jpg" width="200" height="300" alt="kleurrijke paraplu's">
    Lorem ipsum dolor sit amet consectetur adipisicing elit.
    Animi, velit quaerat magnam expedita ipsa eos ...
</p>
<p>Lorem ipsum dolor sit amet consectetur adipisicing elit. Placeat alias eveniet ...</p>
CSS
body {
    max-width: 30em;
    margin: 0 auto;
    font-family: "Trebuchet MS", sans-serif;
    line-height: 1.45;
}

.links {
    float: left;
    margin: 10px 10px 10px 0; /* ruimte rechts en onder de afbeelding */
}

Wat er gebeurt: De afbeelding "zweeft" naar de linkerkant. De omringende tekst vloeit er rechts naast.

Float rechts werkt op dezelfde manier:

CSS
.rechts {
    float: right;
    margin: 10px 0 10px 10px; /* ruimte links en onder de afbeelding */
}

2.2 Een uitgelicht citaat floaten

Een blockquote kan je ook floaten voor een visueel aantrekkelijk effect:

HTML
<p>Lorem ipsum dolor sit amet ...</p>
<blockquote>"To be or not to be, that's the question!"</blockquote>
<p>Facilis non a sapiente quos rerum similique ...</p>
CSS
blockquote {
    margin: 10px 10px 10px 0;
    padding: 10px;
    width: 5em;
    float: left;
    font-family: georgia, serif;
    font-style: italic;
    font-size: 3rem;
    line-height: 1.1;
    color: #444;
    border-top: 3px solid #aaa;
    border-bottom: 3px solid #aaa;
    background-color: #eee;
}
📌 Vuistregel

Bij het floaten van andere elementen dan afbeeldingen moet je altijd een width meegeven, anders weet de browser niet hoe breed het element moet zijn.

2.3 clear — het float-effect stoppen

Soms wil je dat een element niet naast een gefloat element staat, maar eronder. Gebruik daarvoor clear.

CSS
h2 {
    clear: left;  /* geen gefloat element toegelaten links van h2 */
}
Waarde Wat het doet
clear: left Staat niet naast links-gefloate elementen
clear: right Staat niet naast rechts-gefloate elementen
clear: both Staat niet naast enig gefloat element

3. CSS Positioning

Met position kan je elementen op een specifieke plaats zetten. Er zijn vijf waarden.

3.1 position: static — de standaard

Alle elementen hebben standaard position: static. Dit betekent: het element staat gewoon in de normale documentstroom. Je hoeft dit nooit expliciet in te stellen.

3.2 position: relative

Het element blijft in de normale stroom, maar je kan het verschuiven ten opzichte van zijn normale positie.

CSS
.verschoven {
    position: relative;
    top: 20px;   /* 20px naar beneden verschoven */
    left: 10px;  /* 10px naar rechts verschoven */
}
💡 Tip

De lege ruimte blijft behouden — de andere elementen reageren niet op de verschuiving. Dit is zelden direct nuttig, maar het heeft één heel belangrijk doel: het maakt van het element een positioneringscontext voor absolute kinderen (zie hieronder).

3.3 position: absolute

Het element wordt uit de normale stroom gehaald. Het zweeft boven de andere elementen en neemt geen ruimte meer in.

CSS
.badge {
    position: absolute;
    top: 10px;
    right: 10px;
}
📌 Cruciaal

Een absoluut gepositioneerd element wordt geplaatst ten opzichte van zijn dichtstbijzijnde gepositioneerde voorouder. Dat is een voorouder die position: relative, absolute, fixed of sticky heeft.

HTML
<div class="kaart">          <!-- position: relative -->
    <img src="foto.jpg" alt="product">
    <span class="badge">Nieuw</span>   <!-- position: absolute -->
</div>
CSS
.kaart {
    position: relative; /* dit maakt de kaart de positioneringscontext */
}

.badge {
    position: absolute;
    top: 10px;
    right: 10px;
    background-color: red;
    color: white;
    padding: 4px 8px;
    border-radius: 12px;
    font-size: 0.8rem;
}
  • Gebruik dit voor: badges op afbeeldingen, tooltips, overlays, dropdown-menu's.
  • Gebruik dit NIET voor: de algemene lay-out van je pagina.

3.4 position: fixed

Het element wordt geplaatst ten opzichte van het browservenster en scrolt niet mee met de pagina.

CSS
nav.sticky-nav {
    position: fixed;
    top: 0;
    left: 0;
    width: 100%;
    background-color: #222;
    z-index: 100; /* zorgt dat de nav boven andere elementen staat */
}

Gebruik dit voor: navigatiebalken die altijd bovenaan zichtbaar blijven, cookiebanners onderaan het scherm.

⚠️ Let op

fixed elementen overlappen andere inhoud. Voeg een padding-top toe aan body gelijk aan de hoogte van de navigatie, anders verdwijnt de bovenkant van je inhoud achter de navigatiebalk.

3.5 position: sticky

sticky is een combinatie van relative en fixed: het element scrollt mee tot het een bepaalde grens bereikt, dan "kleeft" het vast.

CSS
nav {
    position: sticky;
    top: 0; /* kleeft aan de bovenkant zodra het die bereikt bij scrollen */
    background-color: #222;
    z-index: 100;
}

Dit is de modernste en meest gebruikte manier voor een navigatiebalk die bovenaan blijft plakken.

4. z-index — de stapelvolgorde

Als elementen elkaar overlappen (door position: absolute, fixed of sticky), bepaalt z-index welk element bovenop staat. Hogere waarden komen voor lagere waarden.

CSS
.achtergrond {
    position: absolute;
    z-index: 1;
}

.voorgrond {
    position: absolute;
    z-index: 10; /* staat voor .achtergrond */
}

.overlay {
    position: fixed;
    z-index: 1000; /* staat boven alles */
}
💡 Tip

Gebruik geen willekeurige grote getallen (zoals z-index: 9999). Werk met logische stappen: 1, 10, 100, 1000 voor lagen.

z-index werkt alleen op gepositioneerde elementen (alles behalve static).

5. Overzicht: wanneer wat gebruiken?

Situatie Oplossing
Tekst om een afbeelding laten lopen float
Citaat naast tekst float
Elementen naast elkaar (navigatie, kaartjes) Flexbox
Paginalay-out met kolommen en rijen Grid
Badge op een afbeelding position: absolute
Navigatiebalk die bovenaan blijft position: sticky of fixed
Modal/overlay over de pagina position: fixed
Element verschuiven zonder stroom te verstoren position: relative

positionrelativeVerschuifbaar + positioneringscontext

Onthoud: Float en position zijn voor speciale gevallen. Voor de algemene lay-out van je pagina gebruik je altijd Flexbox of Grid.

Oefeningen

Oefening 1 Artikel met float ADB10

Maak een HTML-pagina artikel.html met:

  • Een hoofding
  • Twee alinea's tekst
  • Tussen de eerste en tweede alinea: een afbeelding die links zweeft met tekst ernaast

Tip: Gebruik float: left op de afbeelding en voeg een marge toe aan de rechterkant zodat de tekst niet direct aansluit.

Oefening 2 Productenkaartje met badge ADB10

Maak een productenkaartje met een "Nieuw"-badge in de rechterbovenhoek van de productafbeelding.

Tip: Gebruik position: relative op de kaart-container en position: absolute op de badge. Stel top en right in om de badge te positioneren.

Huistaak

📋 Huistaak — H14 Float en Positioning

Opdracht: Lay-outpagina met positioning

Maak een pagina layout.html met bijhorend layout.css. Je oefent position-eigenschappen in een realistische context.

Deel 1 — Sticky navigatie (3 punten)

  • Een navigatiebalk bovenaan die plakt terwijl je scrollt: position: sticky; top: 0;
  • De navigatie heeft een achtergrondkleur en een subtiele schaduw (box-shadow)
  • De pagina heeft genoeg inhoud om te scrollen (bv. 4 lange secties)

Deel 2 — Badge op een kaart (3 punten)

  • Een of meerdere kaarten met een badge in een hoek: bv. "Nieuw" of "Uitverkocht"
  • De kaart heeft position: relative, de badge heeft position: absolute; top: 8px; right: 8px;

Deel 3 — Vaste "Terug naar boven"-knop (4 punten)

  • Een knop rechtsonder die altijd zichtbaar blijft: position: fixed; bottom: 24px; right: 24px;
  • De knop linkt naar de bovenkant van de pagina (href="#")
  • Geef de knop een cirkelvorm met border-radius: 50% en een opvallende kleur
  • Voeg een :hover-effect toe
