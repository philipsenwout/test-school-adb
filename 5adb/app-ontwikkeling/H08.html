8.1 Wat is CSS?

CSS staat voor Cascading Style Sheets. Het is een declaratieve taal die beschrijft hoe een HTML-document eruitziet in de browser.

Met CSS bepaal je:

  • Waar een element staat op de pagina
  • Welk lettertype gebruikt wordt
  • Hoe groot een element is
  • Welke kleuren voorgrond en achtergrond hebben
  • Of iets in een raster staat

Belangrijk verschil met HTML:

  • HTML beschrijft de inhoud en structuur ("dit is een kop", "dit is een alinea")
  • CSS beschrijft de vormgeving ("koppen zijn blauw en groot", "alinea's hebben deze lettergrootte")

8.2 De CSS-regel

Een CSS-stijlblad bestaat uit CSS-regels. Elke CSS-regel heeft twee delen:

  1. De selector — welk HTML-element wordt gestyled?
  2. Het declaratieblok — hoe ziet het er uit?
CSS
body {
    font-family: georgia, serif;
    color: #000;
}

De onderdelen

OnderdeelVoorbeeldUitleg

Declaratieblok{ ... }Alles tussen de accolades
Eigenschapfont-familyWat wordt gestyled?

8.3 CSS koppelen aan HTML

Er zijn drie manieren om CSS te koppelen aan HTML.

Methode 1 — Extern stijlbestand (aanbevolen)

HTML
<head>
  <meta charset="UTF-8">
  <title>Mijn pagina</title>
  <link rel="stylesheet" href="stijlblad.css">
</head>

Voordelen:

  • HTML en CSS zijn gescheiden — overzichtelijker
  • Alle pagina's van je site delen hetzelfde CSS-bestand → consistente opmaak
  • Het bestand wordt gecached — snellere browse-ervaring
  • Wil je de kleur van alle titels veranderen? Je past het op één plaats aan.

Methode 2 — Embedded stijlblad (soms nuttig)

HTML
<head>
  <style>
    body {
        font-family: georgia, serif;
        color: #000;
    }
  </style>
</head>

Wanneer gebruiken? Als je maar één HTML-bestand hebt (bv. voor een snelle test).

Methode 3 — Inline stijlinformatie (vermijd dit)

HTML
<h1 style="color: red; font-size: 2rem;">Mijn titel</h1>
📌 Vuistregel

Gebruik altijd een extern CSS-bestand voor je projecten.

8.4 Overerving in CSS

In HTML spreken we over ouders en kinderen. Veel CSS-eigenschappen worden overgeërfd van ouder naar kind.

CSS
body {
    font-family: 'Arial', sans-serif;
    color: #333;
}

Alle tekst op je pagina — titels, alinea's, lijsten — gebruikt nu Arial en de kleur #333, tenzij je dat elders overschrijft.

8.5 Selectoren

Element-selector

CSS
h1 {
    color: navy;
}

p {
    line-height: 1.6;
}

Class-selector (.)

HTML
<p class="update">Het restaurant is gesloten wegens verbouwing.</p>
CSS
.update {
    border: 2px solid orange;
    background-color: #fff3cd;
}
💡 Tip

Geef klassen namen op basis van betekenis, niet op basis van vormgeving. Goed: class="update", class="foutmelding". Slecht: class="rode-rand", class="groene-tekst".

ID-selector (#)

CSS
#hoofdmenu {
    background-color: #1a1a2e;
    padding: 1rem;
}
SelectorVoorbeeldHoeveel keer te gebruiken?

Class.update { }Onbeperkt
ID#hoofdmenu { }Eén keer per pagina

8.6 Cascade en specificiteit

De C van CSS staat voor Cascade (waterval). Het cascade-systeem regelt wat er gebeurt als meerdere CSS-regels van toepassing zijn op hetzelfde element.

8.7 Developer tools

Elke browser heeft ingebouwde developer tools. Je opent ze door:

  • Rechts te klikken op een element → "Inspecteren"
  • Of met de toetsencombinatie F12

Met de developer tools kan je:

  • De HTML-structuur van een pagina bekijken
  • Zien welke CSS-regels van toepassing zijn op een element
  • Zien welke CSS-regels doorstreept zijn (overschreven door een specifiekere regel)
  • CSS tijdelijk aanpassen om te experimenteren

Oefeningen

ADB11 Oefening 1 — Eerste CSS-bestand

Maak een HTML-bestand index.html en een CSS-bestand stijlblad.css. Koppel ze via <link>.

Stijl de pagina:

  • body: lettertype Arial of sans-serif, tekstkleur #333, achtergrond #f5f5f5
  • h1: kleur marineblauw (#003087), grootte 2rem
  • p: regelafstand 1.7
  • .uitgelicht: een gele achtergrond en een lichtoranje rand

Maak een HTML-pagina met twee alinea's, waarvan de eerste de klasse uitgelicht heeft.

ADB11 Oefening 2 — Specificiteit voorspellen

Gegeven de volgende HTML en CSS. Voorspel de kleur van elk element zonder de pagina in de browser te openen. Controleer daarna je antwoord.

HTML
<style>
  p { color: blue; }
  .rood { color: red; }
  #speciaal { color: green; }
  div p { color: orange; }
</style>

<div>
  <p>Alinea 1</p>
  <p class="rood">Alinea 2</p>
  <p id="speciaal">Alinea 3</p>
  <p id="speciaal" class="rood">Alinea 4</p>
</div>

📋 Huistaak — H08 Introductie CSS

Opdracht: Stijl geven aan de receptenpagina

Je koppelt voor het eerst een extern CSS-bestand aan je recept.html van H02 (of een soortgelijke pagina) en schrijft je eerste CSS-regels.

Startbestand

Gebruik je recept.html van H02, of maak een nieuw bestand recept.html met minstens:

  • Een <h1>, twee <h2>-elementen, twee <p>-elementen en een <ul>.

Deel 1 — CSS koppelen (3 punten)

  1. Maak een nieuw bestand stijl.css aan in dezelfde map als je HTML-bestand.
  2. Koppel het CSS-bestand correct aan je HTML via een <link>-element in de <head>.
  3. Voeg ook één inline stijl toe aan een <p>-element: style="font-style: italic;" — alleen om te laten zien dat je het verschil kent.
  4. Voeg ook één interne stijl toe in een <style>-tag in de <head>: body { margin: 0; } — en leg in een commentaar uit waarom je normaal GEEN interne stijlen gebruikt.

Deel 2 — CSS-regels schrijven (4 punten)

Schrijf de volgende regels in stijl.css:

  1. Geef de <body> een font-family: Arial, sans-serif;
  2. Geef de <h1> een color naar keuze (gebruik een kleurwoord, bv. darkblue)
  3. Geef alle <h2>-elementen een color naar keuze
  4. Geef de <p>-elementen een line-height: 1.6;
  5. Geef de <ul> een background-color naar keuze
