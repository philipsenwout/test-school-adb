9.1 Kleuren specificeren in CSS

Er zijn meerdere manieren om een kleur aan te geven in CSS. We bespreken de vier meest gebruikte.

Methode 1 — Kleurnamen

CSS
h1 { color: navy; }
p { color: darkgray; }
button { background-color: tomato; }
💡 Tip

Gebruik kleurnamen zo weinig mogelijk in een echt project. Ze zijn handig voor snelle experimenten, maar je hebt veel minder controle dan met hex of RGB.

Methode 2 — Hexadecimale notatie

Dit is de meest gebruikte methode. Een hex-kleur begint met # gevolgd door zes hexadecimale cijfers (0–9 en a–f):

CSS
h1 { color: #003087; }        /* donkerblauw */
p { color: #333333; }         /* donkergrijs */
nav { background-color: #1a1a2e; } /* bijna zwart */

De zes cijfers zijn drie paren: RR GG BB (rood, groen, blauw).

KleurHexUitleg
Zwart#000000Geen rood, groen of blauw
Wit#ffffffMaximaal rood, groen en blauw
Rood#ff0000Maximaal rood, geen groen, geen blauw

Methode 3 — RGB-decimalen

CSS
h1 { color: rgb(0, 48, 135); }       /* donkerblauw */
p { color: rgb(51, 51, 51); }        /* donkergrijs */
.overlay { background-color: rgba(0, 0, 0, 0.5); } /* 50% transparant zwart */

Methode 4 — HSL (Hue, Saturation, Lightness)

CSS
h1 { color: hsl(220, 95%, 26%); }   /* donkerblauw */
.success { color: hsl(120, 60%, 35%); } /* donkergroen */
.warning { color: hsl(40, 100%, 50%); } /* oranje */

HSL is handig als je variaties van dezelfde kleur wil maken — je verhoogt of verlaagt gewoon de Lightness.

9.2 Kleur en toegankelijkheid

Contrastverhouding

Tekst moet voldoende contrasteren met de achtergrond. De WCAG-norm stelt:

  • Gewone tekst: minimale contrastverhouding van 4,5:1
  • Grote tekst (18pt of 14pt vet): minimale verhouding van 3:1

Gebruik de gratis tool Colour Contrast Analyser (tpgi.com) of de online tool op webaim.org/resources/contrastchecker.

Kleurenblindheid

Wist je dat ongeveer 8% van de mannelijke bevolking last heeft van een verstoord kleurenzicht? Groen-rood kleurenblindheid is de meest voorkomende vorm.

Praktisch gevolg: Gebruik nooit alleen kleur om informatie over te brengen.

  • Foutieve velden: rode rand én een foutpictogram én een foutboodschap in tekst
  • Grafiek: verschillende kleuren én verschillende lijnpatronen
  • Links in tekst: onderstreept (niet alleen een andere kleur)

9.3 Achtergrondafbeeldingen

CSS
body {
    background-image: url('../../images/achtergrond.jpg');
    background-repeat: no-repeat;
    background-size: cover;
    background-position: center;
}
EigenschapWaardenUitleg
background-imageurl('pad')Welke afbeelding?
background-repeatno-repeat, repeat, ...Herhalen?
background-sizecover, contain, pixels, %Hoe groot?
background-positioncenter, top left, ...Waar positioneren?

Gevaar: tekst op achtergrondafbeelding

Achtergrondafbeeldingen achter tekst zijn heel moeilijk leesbaar als de foto zowel lichte als donkere delen heeft.

Gebruik een semi-transparante overlay:

CSS
.hero::before {
    content: '';
    position: absolute;
    inset: 0;
    background-color: rgba(0, 0, 0, 0.5); /* donkere overlay */
}

9.4 Gradiënten

Een gradiënt is een vloeiende overgang tussen twee of meer kleuren. Je maakt ze puur in CSS.

Lineaire gradiënt

CSS
/* Van links naar rechts */
.hero {
    background: linear-gradient(to right, #003087, #0057b8);
}

/* Met richting in graden */
.banner {
    background: linear-gradient(135deg, #1a1a2e, #16213e, #0f3460);
}

/* Met transparantie — handig over een foto */
.overlay {
    background: linear-gradient(
        to bottom,
        rgba(0, 0, 0, 0),
        rgba(0, 0, 0, 0.7)
    );
}

Radiale gradiënt

CSS
.knop {
    background: radial-gradient(circle, #0057b8, #003087);
}

9.5 CSS custom properties (variabelen)

CSS custom properties (ook "variabelen" genoemd) laten je toe waarden op één centrale plek te definiëren:

CSS
:root {
    --kleur-primair: #003087;
    --kleur-secundair: #0057b8;
    --kleur-accent: #ff6b35;
    --kleur-tekst: #333333;
    --kleur-achtergrond: #f8f9fa;
    --lettertype-hoofd: 'Arial', sans-serif;
    --afronding: 8px;
}
CSS
body {
    font-family: var(--lettertype-hoofd);
    color: var(--kleur-tekst);
    background-color: var(--kleur-achtergrond);
}

h1 {
    color: var(--kleur-primair);
}

.knop {
    background-color: var(--kleur-accent);
    border-radius: var(--afronding);
}

Voordelen:

  • Verander een kleur op één plek → de hele site past mee aan
  • Je stijlblad is leesbaarder (--kleur-primair is duidelijker dan #003087)
  • Makkelijker om een dark mode te implementeren

Dark mode met custom properties

CSS
:root {
    --kleur-tekst: #333333;
    --kleur-achtergrond: #ffffff;
}

@media (prefers-color-scheme: dark) {
    :root {
        --kleur-tekst: #eeeeee;
        --kleur-achtergrond: #1a1a2e;
    }
}

body {
    color: var(--kleur-tekst);
    background-color: var(--kleur-achtergrond);
}

9.6 Kleurenschema baseren op een logo — met mate!

De meest voorkomende fout: beginners gebruiken de logokleuren overal, met een overdonderend resultaat.

Praktisch advies:

  • Kies één accentkleur uit het logo
  • Gebruik die voor knoppen, actieve navigatielinks, highlights
  • De rest van je pagina is neutraal (wit, lichtgrijs, donkergrijs, zwart)
CSS
:root {
    --kleur-accent: #e63946;   /* sparingzaam gebruiken */
    --kleur-tekst: #1d3557;    /* donkerblauw voor tekst */
    --kleur-achtergrond: #f1faee; /* bijna wit */
}

Oefeningen

ADB11 Oefening 1 — Kleurenpalet

Maak een HTML-pagina met CSS custom properties. Definieer minstens 4 kleurvariabelen in :root. Gebruik ze voor:

  • Achtergrondkleur van de pagina
  • Tekstkleur
  • Kleur van titels
  • Achtergrond van de navigatie

Controleer met een contrastchecker of je tekst voldoende contrast heeft.

ADB11 Oefening 2 — Gradiënt hero-sectie

Maak een <header> met de klasse hero. Geef deze:

  • Een donkere achtergrondgradiënt (van donkerblauw naar paars, van links naar rechts)
  • Witte tekst
  • Een minimumhoogte van 40vh
  • Gecentreerde tekst

Huistaak

📋 Huistaak — H09 Kleur en Achtergrond

Opdracht: Persoonlijke kleurenpagina

Maak een HTML-pagina kleuren.html met een bijhorend CSS-bestand kleuren.css over een onderwerp naar keuze (muziek, sport, natuur, …). De pagina toont jouw kleurenpalet op een creatieve manier.

Deel 1 — Kleurenpalet (4 punten)

  • Gebruik minstens 4 verschillende kleuren op de pagina
  • Gebruik elke kleurnotatie minstens één keer: een kleurnaam, een hex-waarde, rgb() en hsl()
  • Zorg dat tekst en achtergrond voldoende contrast hebben (controleer met de DevTools)

Deel 2 — Achtergrond en gradient (3 punten)

  • Maak een hero-sectie (een brede banner bovenaan) met een linear-gradient als achtergrond
  • De gradient bevat minstens twee kleuren en de tekst erop is goed leesbaar
  • Voeg ergens een subtiele background-color toe met rgba() (bv. voor een overlay of kaartje)

Deel 3 — CSS custom properties (3 punten)

  • Definieer je hoofdkleuren als CSS-variabelen bovenaan je CSS: :root { --kleur-primair: …; }
  • Gebruik die variabelen consequent door je volledige CSS-bestand
  • Bewijs dat het werkt: wijzig één variabele en controleer dat meerdere elementen meteen van kleur veranderen
